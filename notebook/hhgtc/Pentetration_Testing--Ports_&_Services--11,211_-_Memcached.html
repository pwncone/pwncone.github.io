# Port 11,211 - Memcached


https://memcached.org/
Memcached caches share memory on all of your running systems/servers to create a large pool of spare memory for applications to use.

The interesting stuff to pentesters is the Key + Value pairs on the memcached server.
You might be able to grab names, emails, passwords, etc.
