My custom (some deafult) keyboard shortcuts:
Ctrl + Shift + H - split horizontally
Ctrl + Shift + I - split vertically
Ctrl + Shift + M - maximise window (zoom)
Ctrl + Shift + O - close window (obliterate)

Alt + I - focus the terminal above
Alt + J - focus the terminal left
Alt + K - focus the terminal below
Alt + L - focus the terminal right

Ctrl + Shift + F - search terminal window

Ctrl + + - increase font size
Ctrl + - - decrease font size
Ctrl + 0 - restore original font size
