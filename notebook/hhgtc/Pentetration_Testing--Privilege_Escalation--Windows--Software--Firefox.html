# Locations


### User Profiles


On Windows, Firefox stores user data in profiles in 1 or both (often both) of these locations
• C:\Users\Chase\AppData\Roaming\Mozilla\Firefox\Profiles\<profile>.default
• C:\Users\Chase\AppData\Local\Mozilla\Firefox\Profiles\<profile>.default

## Extract Cookies & Logins from a Backup User Session


• https://stackoverflow.com/questions/19486161/is-there-a-way-to-extract-cookies-from-the-firefox-cookies-file

Examples
• hackthebox Heist

Whilst you browse, Firefox keeps a recovery file storing your cookies, tabs, etc. of your current browsing session.
If a user's browsing session crashes, this file is used to restore what user's session.

The file is called recovery.jsonlz4 and is stored in the user's Firefox profile in a sessionstore-backups folder.

C:\Users\bob\AppData\Roaming\Mozilla\Firefox\Profiles\77nc64t5.default\sessionstore-backups\recovery.jsonlz4

If you read the recovery file, you can potentially find some passwords or other interesting information.
