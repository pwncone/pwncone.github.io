#C++ Introduction


• Dennis Ritchie developed C in 1972
• C++ is an extension of C
• Was developed by Dr Bjarne Stroustrup between 1983-1985 while working at AT&T Bell Labs in New Jersey
• He basically added classes to C, along with a bunch of other stuff - and made an object oriented programming language - C++
• ++ is a bit of a programmer joke (++ increments something by 1)
• C++ is platform independent (unlike C) so you can write programs for any OS
• C++ allows you to create fast, compact programs that are robust and portable

• As C++ gained in populariry, programmers began to make their own extensions to the language
• As a result, it became necessary to agree upon on a standardised C++ version that all programmers could adhere to worldwide
• A joint committee of the American Natinal Standards Institute (ANSI) and the Industry Organisation and Standardisation (ISO) defined a standardised C++ version
◇ This version is known as ANSI C++ (and is portable to any playform and development environment)
