# Locally Running Services


Check for services running locally that weren't available externally/orignially.

netstat -panut
nestat -ac 5 (refresh every 5 seconds)
ss -nutlp

-p = Show the PID and name of the program to which each socket belongs
-a = Show both listening and non-listening (for TCP this means established connections) sockets
-n = Show numerical addresses instead of trying to determine symbolic host, port or usernames
-u = udp
-t = tcp
