# Internal Recon - of Group Policy



gpresult /R - get group policy of current machine

## PowerView


Get-NetGPO - get list of GPO in current domain
Get-NetGPO | select displayname
Get-NetGPO - ComputerName dcorp-student1.dollarcorp.moneycorp.local

Get-NetGPOGroup - get GPO which use Restricted Groups or groups.xml for interesting users
- won't get any info if no Restricted Groups exist

### Find users via Group Policy


Find-GPOComputerAdmin -ComputerName dcorp-student1.dollarcorp.moneycorp.local - get users which are in a local group of a machine

Find-GPOLocation -UserName student1 -Verbose - get machines where the given user is a member of a specific group

## ActiveDirectory PowerShell Module


GetGPO -All - get list of GPO in current domain
Get-GPResultantSetOfPolicy -ReportType Html -Path C:\Users\Administrator\report.html
