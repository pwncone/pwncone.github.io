# Linux - Commands - Finding files/binaries



## find


Find is slow but thorough.
Has lots of features - you can search for files recursively and using regex

find / -name <filename> 2>/dev/null - search / for files named <filename>, direct all errors to /dev/null
find / -user <username> -name basil 2>/dev/null - find all files in / named basil owned by <username>

## locate


Locate is fast because it uses an internal database.
Update the database with - sudo updatedb

locate <filename>

## which


which searches through the directories that are defined in your $PATH variable. Used for finding binaries.

which <binary>
