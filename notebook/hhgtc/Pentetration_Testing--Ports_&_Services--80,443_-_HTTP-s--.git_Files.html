# .git Files


There's a few things you can do if you've found a .git file.

Download the content of the .git file:
wget -r http://web.com/.git

Or have a tool do it for you:
• Git-Money - https://github.com/dnoiz1/git-money
• DVCS-Pillage - https://github.com/evilpacket/DVCS-Pillage
• GitTools - https://github.com/internetwache/GitTools

View the changes made with git diff (a password might have been removed or something).
You can also search for vulns in commits with git-vuln-finder - https://github.com/cve-search/git-vuln-finder
