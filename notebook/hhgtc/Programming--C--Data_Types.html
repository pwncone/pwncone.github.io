# Data Types


When you declare a variable, you reserve space for it in memory.
The data type is what defines how much space in memory is reserved.

Basic data types
char 'A' A single byte, capable of holding one character
int 100 An integer (a whole number)
float 0.123456 A floating point number to 6 decimal places
double 0.1234567890 A floating point number to 10 decimal places

Extras
void Valueless (no type specified)
wchar_t Wide character

Type Modifiers


You can modify data types.

signed positive or negative numbers
unsigned positive numbers (i.e. no signs e.g. no - or .)
short half-size
long double-size

## Data Type Sizes


Unix
In Unix land, data type sizes are defined by ILP32 and LP64
◇ Original 32-bit systems = ILP32
◇ When 64-bit system introduced, transitioned over to LP64
◇ https://www.ibm.com/support/knowledgecenter/en/SSLTBW_2.3.0/com.ibm.zos.v2r3.cbcpx01/datatypesize64.htm
Windows
Windows uses LLP64
◇ LLP64 means 32-bit code can be compiled without changes on 64-bit systems
◇ But to take advantages of 64-bit computing, you would have to rewrite your code
◇ https://docs.microsoft.com/en-us/cpp/cpp/data-type-ranges?view=vs-2019
◇ https://docs.microsoft.com/en-us/cpp/cpp/fundamental-types-cpp?view=vs-2019
