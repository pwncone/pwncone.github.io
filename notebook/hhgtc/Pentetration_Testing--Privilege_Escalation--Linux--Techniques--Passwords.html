# Passwords


Fairily often you'll find passwords inside of files, usually mysql creds and the like.

## Search for passwords


### Search root


grep --color=auto -rnw '/' -ie "PASSWORD" --color=always 2> /dev/null
find . -type f -exec grep -i -I "PASSWORD" {} /dev/null \;

### /var/www/html


grep -r password /var/www/html

### Search for passwords in memory


strings /dev/mem -n 10 | grep -i PASS

### Find sensitive files named ‘password’


locate password | more

## Old passwords in /etc/security/opasswd


• The /etc/security/opasswd file is used also by pam_cracklib
◇ it keeps the history of old passwords so that the user will not reuse them.
• Treat /etc/security/opasswd file like your /etc/shadow
◇ because it ends up containing user password hashes

cat /etc/security/opasswd

## Find recently edited files


Files that were edited in the last 10 minutes
find / -mmin -10 2>/dev/null | grep -Ev "^/proc"
