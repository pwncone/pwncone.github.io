#define


#define can be used to substitute text or numeric values before assembly and compilation.
#define specifies a macro to be substituted by the preprocessor for each occurence of that macro found in the source code

e.g.
#define BOOK “The Murders in the Rue Morge”
Whenever BOOK is found in the source code, it will be replaced with “The Murders in the Rue Morge” before compilation
cout << BOOK << endl;

#define BOOK "The Murders in the Rue Morge"
#define LINE "****************************"
#define PAGES 39
#define PAGESTART (139 - PAGES)

#include <iostream>
using namespace std;

int main()
{
    cout << LINE << endl;
    cout << BOOK << endl;
    cout << LINE << endl;
    cout << "Pages long: " << PAGES << endl;
    cout << "The story starts at page: " << PAGESTART << endl;
    
    return 0;
}


If you compile this program to save the temporary files:
g++ substitutes.cpp -o substitutes -save-temps

Right at the bottom, you'll be able to see the subsitutions in substitutes.ii:
using namespace std;

int main()
{
    cout << "****************************" << endl;
    cout << "The Murders in the Rue Morge" << endl;
    cout << "****************************" << endl;
    cout << "Pages long: " << 39 << endl;
    cout << "The story starts at page: " << (139 - 39) << endl;

    return 0;
}


Define substitutions from the command line

 - -Dname
You can also define substitutions from the command line using the -Dname option
*NOTE: You have enclose strings with escape sequences - \""<string>"\"

Delete any #define directives in your source code and then:
g++ substitutes2.cpp -o substitutes2 -DBOOK=\""The Murders in the Blue Morgue"\" -DLINE=\""********************"\" -DPAGES=39 -DPAGESTART=100
