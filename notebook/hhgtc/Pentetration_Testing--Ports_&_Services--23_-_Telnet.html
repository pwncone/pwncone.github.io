# Port 23 - Telnet


Telnet is a protocol that allows a user on one computer to log into another computer that is part of the same network. It's old now.
Considered insecure because it doesn't encrypt traffic.
Vulnerabilities available on exploit-db.

## Checklist


• Grab banner & connect
◇ nc 10.11.4.120 23 -v
• Collect info
◇ nmap --script "*telnet* and safe" -p 23 <IP>
• Any default credentials?
• Brute-force

## Config files


/etc/inetd.conf
/etc/xinetd.d/telnet
/etc/xinetd.d/stelnet
