# Linux - tmux


https://www.youtube.com/watch?v=Lqehvpe_djs

Benefit of tmux?
• tmux runs all of your terminal windows in its process
◇ If you accidently close a terminal window, it's not lost
◇ You can simply re-open that closed terminal session via tmux

Start a tmux session
tmux new -s <name>

Prefix key
Ctrl + B - default

Create a new window
(prefix) → C

Rename a window
(prefix) → ,

Vertical split
(prefix) → %

Horizontal split
(prefix) → "

Move between split windows
(prefix) → arrow keys
