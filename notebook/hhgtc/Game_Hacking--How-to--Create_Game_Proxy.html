# Create Game Proxy



## Redirecting the game to your proxy


### Add a windows route


route print if # printf your network interfaces
route add [destination] MASK [subnet mask] [proxy to] IF [interface number]
route add 192.223.31.2 MASK 255.255.255.255 127.0.0.1 IF 17

route print # check if your route has been added
route delete [destination]


### If game is using hostname, use

/etc/hosts


Add an entry into /etc/hosts to redirect the game to localhost
127.0.0.1 videogame.com

Ping videogame.com to test that it resolves to 127.0.0.1
ping videogame.com
