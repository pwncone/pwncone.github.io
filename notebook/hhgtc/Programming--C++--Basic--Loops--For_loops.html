#For loops


For loops are used to repeatedly cycle through something a specified number of times
e.g. for every letter in a word, print letter
e.g. for every number in range 1 - 1000, print number

This for loop will print out all the numbers from 1 to 10.
#include <iostream>
using namespace std;

int main() {
    for(int i = 0; i < 10; i++) {
        cout << i << endl;
    }

    return 0;
}


for(int i = 0; i < 10; i++) translates to:
i = 0, whilst i is less than 10, run the block of code below, and increment i by 1

pseudocode:
for (init; condition; increment) {
code goes here;
}
