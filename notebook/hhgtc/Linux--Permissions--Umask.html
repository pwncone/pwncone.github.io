# Linux - Permissions - Umask


• Every file that gets created comes with a default set of permissions.
• You can change the default set of permissions with the umask command
• umask takes away permissions, starting from 777
• If you run umask, it will change the default set of permissions of new files you create for the current session
• If you want to change your umask change to be perisistent, you have to modify your startup file - .profile

The default umask on most distributions is 022
◇ users have 7 - read/write/execute
◇ groups have 5 - read/execute
◇ others have 5 - read/execute

$ umask 021
