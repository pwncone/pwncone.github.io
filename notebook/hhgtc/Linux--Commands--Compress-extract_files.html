# Linux - Commands - Compress & extract files



## tar


• tar creates archives

tar -cvf file.tar <file1> <file2> - create tar archive
◇ c - create
◇ v - verbose
◇ f - the filename of the tar file
tar czf myfile.tar.gz - create a compressed tar archive
◇ z - use gzip to compress

tar -xf file.tar - extract tar archive
tar xfz file.tar - extract compressed tar archive
◇ z - use gunzip to decompress

tar -tf file.tar - list contents of archive

## gzip


• gzip compresses archive/files
• gzip can't add multiple files into one archive

gzip <file> - compress provided file and rename it <file>.gz
gunzip mycoolfile.gz - decompress/extract gzip archive
gzip -d file.gz - decrompress/extract gzip archive
