#Classes


A class is a data structure that can contain both variables and functions.
The functions are known as methods, and the variables/functions collectively are known as members.

Classes are used to create virtual objects.
Objects in the real world are defined by their attributes and actions
e.g. a dog has attributes like age, weight, breed and an action it does is bark

class Dog
{
 private:
    int age, weight;
    string breed;
 public:
    void bark();
};


A class declaration only defines a data structure.
In order to create an object you must declare an ‘instance’ of that data structure.

Dog fido;


Alternatively, you can declare an instance/multiple instances before the final semicolon when declaring a class

class Dog
{
 private:
    int age, weight;
    string breed;
 public:
    void bark();
} fido, dave, pom;


Access Specifiers


Access to class memebrs from outside the class is controlled by 'access specifiers'.
Access specifiers can be 1 of the 3 keywords:

public - public members are accessible from any place where the class is visible
private - private members are accessible only to other members of the same class
protected - protected members are accessible only to other members of the same class and to memebers fo classes drived from that class

By default, all class members are private.
Any members that appear in the class declaration without an access specifier will have private access.
