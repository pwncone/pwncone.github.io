# Linux - Terminals


In Linux, there are 2 types of terminals.
• regular terminals - tty
• psuedoterminals - pty

## Regular Terminal


• Uses text for input and output
• Everything you type interfaces directly with the computer
• It's the old fashioned system of:
◇ 1 mainframe computer
◇ Multiple terminals so multiple people can use the computer at once
◇ Each terminal has its own login etc.

## Pseudo-Terminal


• Psuedo-terminals emulate terminals
• Uses text for input and output
◇ BUT DIRECTS all of that input/output to another program
▪ e.g. you ssh into a box- you're a on a pseudo-terminal
- you ls, and that ls command sends its output to a pseudo-terminal which is attached to your ssh connection
• Every terminal you use on an OS nowadays will be a pseudo-terminal
