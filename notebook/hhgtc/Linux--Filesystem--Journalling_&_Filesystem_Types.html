# Linux - Filesystem Types


A file system (or filesystem) is used to control how data is stored and retrieved
Without a file system, there would be no way to tell where one piece of data stops and the next begins.

• There are lots of different filesystem implementations available
◇ Some are faster than others, some support larger capacity storage etc.
• Different filesystems have different ways of organizing their data
• Because there's many different filesystems implemenetions
◇ applications need a way to deal with the different operations
◇ To solve this, there's the Virtual File System (VFS) abstraction layer
◇ The VFS sits between applications and the filesystem and interfaces the two
▪ Therefore, no matter what filesystem you have, your applications will work with your filesystem
• Each parition can have a different filesytstem

## Journalling


• Journalling logs changes to your filesystem to a log file just before they happen
◇ .e.g you transfer a file, the log file (journal) notes down what's going to happen
• When the file transfer is finished, the journal marks the task as complete
• Journalling improves reliability of filesystems by logging filesystem updates
• If a crash happens, the disk knows which files have been completed against the disk and which haven't

## Common Desktop Filesystem Types


### ext4


◇ ext4 is what most modern Linux filesystems are nowadays
◇ stands for Fourth Extended Filesystem
◇ supports 1 exbabyte volumes
◇ supports 16tb files
◇ backwards compatible with ext2 and ext3
◇ can pre-allocate space on disk for files using fallocate() system call (allows for media streaming/databases)
◇ unlimited number of subdirectories
◇ journalling

### Btrfs


• Btrfs stands for B-Tree File System
• Btrfs is a new filesystem for Linux
◇ supports snapshots
◇ incremental backups
◇ has performance increases
• It's widely available, but not quite stable yet

### XFS


• XFS is a high performance journaling file system
• Great for a system with large files, such as a media server

### NTFS and FAT


• NTFS and FAT are Windows filesystems

### HFS+


• HFS+ is a Macintosh filesystem
