# Linux - Users - /etc/shadow (passwords)


/etc/shadow file is used to store information about user authentication.
Requires root to read.

The contents is very similar to /etc/passwd, except that an encrypted password is included.
root:MyEPTEa$6Nonsense:15000:0:99999:7:::
1:2:3:4:5:6:7:8:9

1. Username
2. Encrypted password
3. Date of last time password changed
◇ Expressed as the number of days since Jan 1, 1970 - linux epoch time
◇ If 0, this means the user should change their password the next time they log in
4. Minimum password age
◇ Days that a user will have to wait before being able to change their password again
5. Maximum password age
◇ Maximum number of days before a user has to change their password
6. Password warning period
◇ Number of days before a password is going to expire
7. Password inactivity period
◇ Number of days after a password has expired to allow login with their password
8. Account expiration date
◇ date that user will not be able to login
9. Reserved field for future use
