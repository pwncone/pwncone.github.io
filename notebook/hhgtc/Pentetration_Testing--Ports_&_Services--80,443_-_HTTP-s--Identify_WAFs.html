# Identify WAFs


Web App Firewalls - WAFs - parse HTTP traffic and block both ingress queries and egress responses that match known signatures (e.g., SQL injection and XSS strings).

WAFs can be
• built into a web server (e.g. Apache's mod_security module)
• a dedicated appliance
• run as a cloud service

## Identify WAFs


### wafw00f


wafw00f http://website.com

### Nmap


nmap -p 80 --script http-waf-fingerprint www.website.com

## Avoiding WAFs


http://securityidiots.com/Web-Pentest/WAF-Bypass/waf-bypass-guide-part-1.html

An effective WAF avoidance tactic is to route HTTP requests around the security mechanism.
Modify your local /etc/hosts file and evaluate each HTTP/S endpoint to identify routes that bypass the WAF.
