# Recon Scripts


There are scripts that will automate the information gathering process for you.
This is a huge saver of both time and effort.

Do note that using scripts will probably generate a lot of noise.

## lse.sh


https://github.com/diego-treitos/linux-smart-enumeration

A very pretty and thorough enumeration script.
It's pretty much LinEnum.sh but improved.

## linuxprivchecker.py


https://github.com/sleventyeleven/Linuxprivchecker
This requires python to be installed on the system (because it's a python script).

NOTE:
You might have to point to script to your python install. Try running it first, if it doesn't work follow this:

locate python | grep bin - take note of where the python install is
nano linuxprivchecker.py
modify #!/usr/env python to #!<location of python install> e.g. #!usr/bin/python

## LinEnum.sh


https://github.com/rebootuser/LinEnum

./LinEnum.sh -s -r lin-enum.txt -e <output location> -t
-s supply current user password
-r enter report name
-e export location
-t include thorough (lengthy) tests

## unix-privesc-check


Available in Kali: /usr/bin/unix-privesc-check
Serve the file to your victim somehow and run it.

You can grep the output for WARNING
./unix-privesc-check standard/detailed > upc.txt
grep WARNING upc.txt

This program relies on the strings binary.
If it fails, it's probably because strings isnt' available.
