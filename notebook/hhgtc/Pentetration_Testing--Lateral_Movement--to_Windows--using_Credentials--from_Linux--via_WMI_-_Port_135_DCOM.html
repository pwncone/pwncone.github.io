# Using credentials - from Unix - via WMI / port 135 DCOM


Initial access to DCOM is via port 135 DCOM.

## Impacket's wmiexec.py


• Executes commsnds through WMI - Windows Management Instrumentation
• Runs under the user account (has to be Admin), not SYSTEM
• Doesn't generate noisy messages in event log

Needs DCOM / RPC access -> so has to be able to access DCOM ports on target machine

wmiexec.py DOMAIN/username@10.10.10.10
