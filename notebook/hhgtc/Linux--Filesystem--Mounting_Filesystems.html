# Linux - Filesystem - Mounting Filesystems


• Before you can view the contents of your filesystem, you will have to mount it
• To do this you'll need a mount point
◇ A mount point is a directory on the system where the filesystem is going to be attached

1. Create a mount point (a directory)
mkdir /storageDrive
2. Mount the filesystem to the mount point/directory
sudo mount -t ext4 /dev/sdb2 /storageDrive

Unmount a Device


sudo umount /storageDrive
or
sudo umount /dev/sdb2

## Device UUIDs


Sometimes you'll need to be more specific than /dev/sda1 or /storageDrive.
In those cases, we can use the device's UUID (universally unique ID) that the kernel assigns it.

sudo blkid - view device UUIDs on your system
pete@icebox:~$ sudo blkid
/dev/sda1: UUID="130b882f-7d79-436d-a096-1e594c92bb76" TYPE="ext4" 
/dev/sda5: UUID="22c3d34b-467e-467c-b44d-f03803c2c526" TYPE="swap" 
/dev/sda6: UUID="78d203a0-7c18-49bd-9e07-54f44cdb5726" TYPE="xfs" 


Mount using device UUID:
sudo mount UUID=130b882f-7d79-436d-a096-1e594c92bb76 /storageDrive
