# ELF Symbol Table


https://www.tutorialspoint.com/compiler_design/compiler_design_symbol_table
The symbol table is created by the compiler.
It stores information about variables, functions, objects, classes etc.
◇ address
◇ data type

The symbol table can be read with readelf -s <binary>
