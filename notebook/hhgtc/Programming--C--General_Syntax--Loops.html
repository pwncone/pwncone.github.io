# Loops


Repeatedly do something. Loop loop loop.

### Loop directives


#### break
break will break out of a loop.
// break kills the loop after 10 loops
int i = 0;
while (1) {
    i++;
    if (i == 10) 
    {
        break;
    }
}

## for Loops


For thing in thing, do thing

// Loop through numbers 1 to 10 -> print them
for (int i = 0; i < 10; i++)
{
    printf("%d \n", i);
}

// Loop through string -> print each letter
char hey[5] = "Hello";
for (int i = 0; i < strlen(hey); i++)
{
    printf("%c \n", hey[i]);
}


## while Loops


While thing is or isn't true, do thing

// Execute code 5 times
int i = 0;
while (i < 5)
{
    printf("number %d", i);
    i++;
}


## do while Loops


Same as a while loop except that it's guaranteed to execute once.

int a = 10;

do
{
    printf("a = %d \n", a);
    a++;
} while (a <= 20);


## Inifnite loops


Again again again again again again.

// Infinite while loop
while (1)
{
    printf("sandwhich pls \n");
}

// Infinite for loop
for ( ; ; ) 
{
