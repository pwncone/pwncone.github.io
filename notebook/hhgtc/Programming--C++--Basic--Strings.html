#Strings


C++ provides following two types of string representations −
• The C-style character string.
• The string class type introduced with Standard C++ #include <string>

The C-Style Character String


The C-style character string originated within the C language and continues to be supported within C++.
It's actually a one-dimensional array of characters which is terminated by a null character '\0'

Declare and initialise a string:
char greeting[] = "Hello";

An equivalent would be this:
char greeting[6] = {'H', 'e', 'l', 'l', 'o', '\0'};

In memory the string will look like this (remember that a \0 null terminator is inlcuded at the end of the string):


You don't need to write the \0 null terminator yourself, the C++ compiler automatically places the '\0' for you when you initialise the array.

The String Class in C++


The standard C++ library provides a string class type that adds extra functionality.
Strings are stored in arrays still, but they're just easier to create/work with etc. #include <string>

#include <iostream>
#include <string>
using namespace std;

int main () {
   string str1 = "Hello";
   string str2 = "World";
   string str3;
   int len;

   // copy str1 into str3
   str3 = str1;
   cout << "str3 : " << str3 << endl;

   // concatenates str1 and str2
   str3 = str1 + str2;
   cout << "str1 + str2 : " << str3 << endl;

   // total length of str3 after concatenation
   len = str3.size();
   cout << "str3.size() :  " << len << endl;

   return 0;
}


Loop through a string


Looping through a string is just looping through an array.
i = 0, from i up to the size of the string (e.g. 10), print string[i]

#include <iostream>
#include <string>
using namespace std;

int main() {
    string myString = "Hello";
    
    for(int i = 0; i < myString.size(); i++) {
        cout << myString[i] << endl;
    }
    
    return 0;
}
