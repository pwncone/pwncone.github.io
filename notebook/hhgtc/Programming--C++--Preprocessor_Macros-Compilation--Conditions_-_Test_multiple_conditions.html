#Testing Multiple Conditions



Testing Multiple Conditions -

in Headers


For testing multiple conditions, #ifdef can be expressed as #if defined with further tests as #elif defined

#if defined _WIN32
    #define PLATFORM "Windows 32bit"
#elif defined _WOW64
    #define PLATFORM "Windows 64bit"
#elif defined __linux
    #define PLATFORM "Linux"
#else 
    #define PLATFORM "Who knows... I sure don't."
#endif

#include <iostream>
using namespace std;

int main()
{
    cout << PLATFORM << endl;
    
    return 0;
}


Compiler Predefined Macros
Some macros are automatically predefined by the compiler.
Typically, these names begin with underscore __ to avoid accidental confusion with user-specified macros.
The compiler's predefiend macros are platofrm-specific
(e.g. you can run multiple definition tests to identify a user's platform, as above)

To see a list of the compiler's predefined macros, run a command calling the cpp preprocessor directly
touch predefinedMacros.txt
cpp -dM predefinedMacros.txt


Testing Multiple Conditions -

in

Code/Elsewhere


#ifdef and #ifndef can be extended using #elif and #else to provide alternatives.

#include <iostream>
using namespace std;

int main()
{   
    // if WEATHER is defined
    #ifdef WEATHER
        cout << WEATHER << endl;
    // if TEMPERATURE is defined
    #elif TEMPERATURE
        cout << TEMPERATURE << endl;
    // otherwise...
    #else
        cout << "I'm not sure about the weather today :/" << endl;
    #endif
    
    return 0;
}
