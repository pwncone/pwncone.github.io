# Linux - Commands - Process Management



top - list currently running processes
kill <process id> - kill a process (sends SIGTERM signal - a termination request to cleanly release its resources)
kill -9 <process id> - really kill a process (sends a SIGKILL signal)

lsof - list open files and their assosciated processes
fuser - shows information about the process that is using the file
ps m - view process threads
