#While loops


A while loop statement repeatedly executes the code as long as a given condition is true.

pseudocode:
while (condition) {
code goes here;
}

This example will print out the values from 10 to 20
#include <iostream>
using namespace std;
 
int main () {
   // Local variable declaration:
   int a = 10;

   // while loop execution
   while( a != 20 ) {
      cout << "value of a: " << a << endl;
      a++;
   }
 
   return 0;
}
