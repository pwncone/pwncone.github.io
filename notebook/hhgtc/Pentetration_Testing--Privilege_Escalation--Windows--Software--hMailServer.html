# hMailServer


hMailServer is a free open source email server for Windows.
I would suggest getting RDP access so that you can browse the database locally with hMailServer.exe program.

## Locations


### Credentials


C:\Program Files\hMailServer\Bin\hMailServer.INI

The MSSQLCE password is a unique implementation of blowfish.
Decrypt with - https://github.com/GitMirar/hMailDatabasePasswordDecrypter

### Database


C:\Program Files\hMailServer\Database\hMailServer.sdf
Open with https://github.com/christianhelle/sqlcequery

### Read emails


C:\Program Files\hMailServer\Data\
