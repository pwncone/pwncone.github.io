# Info Gathering



## nbtscan


nbtscan is a tool specialised for identifying NetBIOS information (think this is a windows thing)
nbtscan -r 192.168.11.0/24

## Nmap SMB scripts


Run all SMB scripts
nmap -v -p 139,445 --script smb* 192.168.11.236
List all the smb scripts available:
ls -l /usr/share/nmap/scripts/smb*
OS discovery with SMB:
nmap -v -p 139, 445 --script=smb-os-discovery 192.168.11.227
Check for known SMB vulns:
nmap --script smb-vuln* -p 139,445 10.10.10.152
