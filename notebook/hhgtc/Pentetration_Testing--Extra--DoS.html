## SYN Flooding Using hping3


A SYN flood is where the attacker sends repeated SYN requests to target system, which eventually causes a denial of service because the attacker never replies to the server with the expected ACK response and SYN packets just stack up on the server.

hping3 is a network tool able to send custom TCP/IP packets and display target replies
hping3 -S [target IP] -a [from IP] -p 22 --flood
