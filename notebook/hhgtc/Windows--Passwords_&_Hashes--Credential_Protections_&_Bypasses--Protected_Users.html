# Protected Users


https://docs.microsoft.com/en-us/windows-server/security/credentials-protection-and-management/protected-users-security-group

Protected Users is an Active Directory security group.

## Protections


Members of the Protected Users group have the following protections applied:
◇ Prevents cache-ing of plaintext credentials with:
▪ Credential delegation - CredSSP
▪ WDigest
▪ NTLM authentication
▪ Kerberos (after the initial TGT is acquired)
◇ Disables Kerberos DES and RC4 keys
