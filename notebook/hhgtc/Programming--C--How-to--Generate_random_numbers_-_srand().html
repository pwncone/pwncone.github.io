#Tidbits - rand() / Random Numbers



rand()


• rand() is in stdlib.h
• rand() genreates a pseudo-random positive integer
• By default it returns a number between 0 and 32,767

srand()


• Numbers generated by rand() aren't actually random
◇ rand() generates the same sequence of numbers each time the program is run
• To create randomness, we specify a non-static seed value using srand()
• srand(), using its seed value, causes rand() to generate a different sequence of numbers
◇ the default srand() value is 1
• The simplest “non-static” integer/seed to use is the time - srand(time(0))
◇ this means the sequence of numbers generated by rand() will be different each time the program is run

#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main()
{
    int min_value = 100;
    int max_value = 500;
    
    srand(time(0));                                        // seed srand() with the current time
    int random_num1 = rand() % max_value;                  // generate random number from 0 to 500
    int random_num2 = (rand() & max_value) + min_value;    // generate random from 100 to 500

    printf("random_num1 = %d \n", random_num1);
    printf("random_num2 = %d \n", random_num2);

    return 0;
}


• Set a maximum upper limit using the modulus operator - %
rand() % 9 - specify a range from 0 - 9

• Set a minimum lower limit, add the lower limit to the result of rand() modulus upper limit
(rand() % 9) + 1 - specify a range from 1 - 10
