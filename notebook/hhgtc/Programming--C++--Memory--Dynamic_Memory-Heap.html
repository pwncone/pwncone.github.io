#Dynamic Memory


Memory in your C++ program is divided into two parts:
• the Stack − contains the arguments for a function and all the variables declared inside of that function
• the Heap − a free-floating region of memory that can be used to allocate memory dynamically when program runs.

Dynamic memory means working with heap.

Old operators from C
malloc() or calloc() - reserve memory on the heap
realloc() - reseize variables on the heap
free() - delete memory on the heap you're not using anymore

BUT DON'T USE THESE. Instead, use the shiny new C++ operators for this...

new


You can allocate memory at run time within the heap using - new <data-type>;
int* num = NULL;
num = new int;

*num = 4; - store value at assigned memory region

delete


When you feel a variable that has been dynamically allocated is not required anymore use - delete
delete num;

#include <iostream>
using namespace std;

int main()
{
    int* pointerValue = NULL; // pointer intialised with NULL
    pointerValue = new int; // request memory for the variable

    *pointerValue = 4; // store value at memory address
    cout << "Value of num: " << *pointerValue << endl;

    delete pointerValue; // free up memory

    return 0;
}


Dynamic Memory Allocation for Arrays


#include <iostream>
using namespace std;

int main()
{
    int* pointerValue = NULL; // pointer initialised with NULL
    pointerValue = new int[20]; // request memory for the array

    delete [] pointerValue; // delete array pointed to by array

    return 0;
}
