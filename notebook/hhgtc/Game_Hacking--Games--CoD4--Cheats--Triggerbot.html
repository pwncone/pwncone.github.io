You can't use crosshairClientNum in the cg_t struct because it doesn't work at long distances.

## Check if aiming at player


Use the internal game function CG_Trace to check if you're aiming at a player.
The client number of the player you're aiming at will be stored in trace.hitID

If it's an enemy, shoot at them.

## Shoot


You can shoot 1 bullet by writing +attack into the game's console.
Refer to the Internal Functions subnode for
Cmd_ExecuteSingleCommand(1, 1, "+attack\n");
Cmd_ExecuteSingleCommand(1, 1, "-attack\n");


HOWEVER, the version above doesn't work for semi auto weapons
but works perfectly for full auto.

Alternatively, you can do use a bool to turn shooting on/off.
This causes semi-auto guns to rapid fire, but bugs out full auto guns.
static bool is_firing = false;

// If "not firing" but looking at a player, shoot at them
if (is_firing == false && trace.hit_id == entity->client_number && trace.fraction <= 1.0f)
{
is_firing = true;
Cmd_ExecuteSingleCommand(1, 1, "+attack\n");
}
// If player is "firing" but not looking at player anymore, stop shooting
else if (is_firing == true)
{
is_firing = false;
Cmd_ExecuteSingleCommand(1, 1, "-attack\n");
}
