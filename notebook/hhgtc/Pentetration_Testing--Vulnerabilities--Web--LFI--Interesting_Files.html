# LFI - Interesting Files to Check


You could just shove all the files listed here into a wordlist and test them with a script.
Or just test them manually.

## Linux


### Passwords


/etc/passwd
/etc/shadow
/var/backup

### Root


/root/.bash_history

### Mail


/var/mail/root
/var/mail/user

### User


/home/user/.ssh/id_rsa
/home/user/.bash_history

### System info


/etc/issue
/etc/motd
/etc/group
/var/log/dmessage
/var/spool/cron/crontabs/root

### Apache config files


https://wiki.apache.org/httpd/DistrosDefaultLayout
