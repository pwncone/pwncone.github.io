## DWORD

Range is 0 through 4294967295

## TCHAR


https://www.codeproject.com/Articles/76252/What-are-TCHAR-WCHAR-LPSTR-LPWSTR-LPCTSTR-etc
A TCHAR is a generic data type.
It can hold either standard 1 byte ASCII characters - a char
or 2-byte Unicode characters - a wchar_t

Windows uses UNICODE by default.
If you want your code to be independent of character encoding -> use generic data types.
Instead of char or wchar_t, use
TCHAR

BUT DON'T DO THIS. JUST USE CHAR OR WCHAR_T.
