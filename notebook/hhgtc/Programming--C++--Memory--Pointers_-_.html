#Pointers


Pointers are variables that store the memory addresses of other variables (pointers point to the location of other variables)

Declaring/Initialising Pointers


To delcare a variable as a pointer, use * after the data type - int*
You then initialise the pointer by assinging it the memory address of another variable - &a

*NOTE* You must use the same data type for you pointer as you did the variable
e.g. your variable is an int, your pointer must be an int. your variable is a string, your pointer must be a string
int a = 7;
int* firstPointer = &a;

string word = "moon";
string* secondPointer = &word;


Retrieve the address of the pointers
cout << &firstPointer << endl;
cout << &secondPointer < endl;


Retrieve the values in the pointers
i.e. the memory address of the variable you're pointing to
cout << firstPointer << endl;
cout << secondPointer << endl;


Retrieve the data from the variables you're pointing to
cout << *firstPointer << endl;
cout << *secondPointer << endl;
