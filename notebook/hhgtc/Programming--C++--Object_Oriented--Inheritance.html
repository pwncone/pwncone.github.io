#Inheritance


Inheritance is the 2nd principle of Object Oriented Programming (OOP).

A C++ class can be created as a brand new class or it can be derived from an existing class.
A dervied class inherits members (variables and functions) from its parent class in addition to its own.

Being able to inherit members from a base class allows derived classes to share common properties
e.g. a dog base class defines the common characteristics between all dogs, and derived classes define each breed
or
a polygon base class defines width and height properties, and then ‘rectange’ and ‘triangle’ classes could be derived from the polygon base



#include <iostream>
using namespace std;

class Polygon
{
protected:
    int width, height;
public:
    void setValues(int w, int h) {width = w; height = h;}
};

class Rectangle : public Polygon
{
public:
    int area() {return (width * height);}
};

class Triangle : public Polygon
{
public:
    int area() {return ((width * height) / 2);}
};

int main()
{
    Rectangle rect1;
    Triangle tri1;

    rect1.setValues(4, 5);
    tri1.setValues(4, 5);

    cout << "Rectangle area: " << rect1.area() << endl;
    cout << "Triangle area: " << tri1.area() << endl;
    
    return 0;
}
