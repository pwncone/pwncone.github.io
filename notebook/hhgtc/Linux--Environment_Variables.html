# Linux - Environment Variables


Environement variables are system-wide (technically incorrect, but it helps me understand it) variables containing information like the user's home directory, their current working directory, their username, the paths to their binaries etc.
Users can set their own environment variables if they want.

env - view your environment variables
echo $HOME - view path of your home directory
echo $USER - view your username
echo $PATH - view the paths your system searches for binaries

## PATH Environment Variable


One particularly important variable is the PATH Variable.
The PATH variable stores an ordered, colon-separated list of the paths your system searches for binaries/commands.

$ echo $PATH
/usr/local/sbin:/usr/local/bin:/usr/sbin:/bin

In the PATH example above, if you type in cat, your system will :
- first search /usr/local/sbin for cat,
- then /usr/local/bin,
- then /usr/sbin
- etc.
