# Grab Protected Files - Copy-VSS.ps1


https://github.com/samratashok/nishang/blob/master/Gather/Copy-VSS.ps1

The script must be run from an elevated shell.

• Uses the VSS service (starts it if not running) to create a shadow of C:
◇ copies the SAM file
◇ if run on a Domain Controller -> ntds.dit and SYSTEM hive are also copied.
▪ Sometimes the ntds.dit is present in other locations like
- D:\NTDS or C:\Windows\NTDS
- Use $ntdsSource variable to provide the directory.

Import-Module .\Copy-VSS.ps1
Copy-VSS -DestinationDir C:\Users\Public\nishang-vss\
