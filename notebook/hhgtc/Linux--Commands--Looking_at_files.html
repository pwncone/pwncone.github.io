# Linux - Commands - Looking at files



ls - List files in directory
ls -ltr - Sort list by last modified. -time -reverse
ls -alh - show all files, list format, human readable files sizes

file - Show info about file
cat - Read/output content of a file
more - read a file 1 terminal page at a time
less - the same as more but better. use less :)
Use /<searchterm> to search. n to scroll to next search result, q to quit.
head - output the first 10 lines of a file
tail - output the last 10 lines of a file
tail -f - output last 10 lines and print more lines as the file grows
wc -l <filename> - get the line count of a file
