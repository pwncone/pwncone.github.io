#Creating multiple objects


A program can create multiple objects by declaring multiple instances of a class.
Each object can have unique attributes by assigning individual values via the class's setter methods.

It's often convenient to combine the setter methods into a single method that accepts arguments for each private variable.
This means that all values can be assigned with a single staement.

Prototype


If a method has more than 2 lines, it should be declared as a ‘prototype’ - after the class declaration (not inline).
In the class:
public:
// SETTER METHODS
void setValues(int, int, string);

After the class:
// definition block for protoype to identify the class in which it resides
void Dog::setValues(int age, int weight, string colour)
{
this ->age = age;
this ->weight = weight;
this ->colour = colour;
}

#include <iostream>
#include <string>
using namespace std;

class Dog
{
private:
    // CLASS-ONLY DOG ATTRIBUTES
    int age, weight;
    string colour;
public:
    // SETTER METHODS
    void setValues(int, int, string);

    // GETTER METHODS
    int getAge() {return age;}
    int getWeight() {return weight;}
    string getColour() {return colour;}

    // PROGRAM-WIDE DOG ACTIONS
    void bark() {cout << "WOOF!" << endl;}
};

// definition block for protoype to identify the class in which it resides
void Dog::setValues(int age, int weight, string colour)
{
    this ->age = age;
    this ->weight = weight;
    this ->colour = colour;
}

int main()
{
    Dog fido;
    fido.setValues(6, 40, "brown");
    Dog pom;
    pom.setValues(10, 15, "grey");

    cout << "Fido is " << fido.getAge() << " years old." << endl;
    cout << "Fido is " << fido.getWeight() << " pounds heavy." << endl;
    cout << "Fido is a " << fido.getColour() << " dog." << endl;

    cout << endl;
    fido.bark();
    cout << endl;

    cout << "Pom is " << pom.getAge() << " years old." << endl;
    cout << "Pom is " << pom.getWeight() << " pounds heavy." << endl;
    cout << "Pom is a " << pom.getColour() << " dog." << endl;
    
    cout << endl;
    fido.bark();

    return 0;
}
