# Linux - Users & Groups - Permissions - Changing file ownership


Files by default are owned by the user who creates them + that user's default group.

## chown


chown <user>:<group> <file> - change ownership of file
chown bob:marketing plans.txt

chown <user to own the file> <file> - change the owner of the file without changing the group
chown jane plans.txt

To change ownership of directory and files in it, use the recursive option -R
chwon -R bob:marketing /var/www/
