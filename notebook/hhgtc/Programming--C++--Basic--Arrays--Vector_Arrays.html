#Vector Arrays


A vector array is an alterntive to a regular array, and has the advantage that its size can be changed as the program requires.

Vectory library


In order to use vectors in your C++ program, you have include the vector library
#include <vector>


These are the predefined functions in the vector library:
at(position) Gets the value contained in the specified element number
front() Gets the value in the first element
back() Gets the value in the final element
clear() Removes all vector elements
empty() Returns true (1) if the vectory is empty, and returns false (0) otherwise
size() Gets the number of elements in the vectory
pop_back() Removes the final element
push_back(value) Adds an element to the end of the vector (the value you specify)

Declaring a vector


vector <data-type> vector-name (size)
e.g.
vector <int> myVector1(3);


An int vector will, by default, have each element automatically initialised with a 0 value
Alternatively, you can specify a different initial value:
vector <data-type> vector-name (size, initial-value)
e.g.
vector <int> myVector2(3, 100);

This will declare and initialise a vector array of 3 elements each with a value of 100
