#Do while loops


A do...while loop is similar to a while loop, except that a do...while loop is guaranteed to execute at least one time because it will execute the do block first
Use do while loops when you want your loop to execute at least once.

It executes the do block first, then checks the condition
#include <iostream>
using namespace std;
 
int main () {
   // Local variable declaration:
   int a = 10;

   // do loop execution
   do {
      cout << "value of a: " << a << endl;
      a = a + 1;
   } while( a < 20 );
 
   return 0;
}
