#Elastix - SIPVicious


SIPVicious suite is a set of security tools that can be used to audit SIP based VoIP systems.
https://github.com/EnableSecurity/sipvicious

WARNING


THIS SHIT CAN MAKE PHONES RING.
IT'S NOT EXACTLY STEALTHY.

Verify a PBX is running on the server with svmap
root@gotham:~# svmap 10.10.10.7
| SIP Device      | User Agent          | Fingerprint |
-------------------------------------------------------
| 10.10.10.7:5060 | FPBX-2.8.1(1.8.7.0) | disabled    |


svwar is the war dialing part of SIPVicious
svwar -e100-1000 10.10.10.7
svwar --force -e100-1000 10.10.10.7

Try changing the method you're using with -m
This is like Nmap and TCP scans - full scan? connect scan? etc.
svwar -m INVITE -e100-1000 10.10.10.7 - very likely to cause phones to ring!
