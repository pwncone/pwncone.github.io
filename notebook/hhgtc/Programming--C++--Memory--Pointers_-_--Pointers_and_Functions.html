#Pointers and Functions



Pointers as Arguments to Functions


Pointers can be passed as arguments to functions.
When pointers are passed as arguments, the data gets passed “by reference” - i.e. they pass the data in the memory address that it's pointing to

e.g.
int num = 5;
int* tmpPtr = &num;
tmpPtr points to the memory address of num
When you pass tmpPtr as an argument to a function, the data inside of the memory address that it's pointing to gets passed, which in this case will be 5

#include <iostream>
using namespace std;

void writeOutput(int* value)
{
    // write out data pointed to by value
    cout << "Current value: " << *value << endl;
}

void computeTriple(int* value)
{
    // multiply data pointed to by value by 3
    *value *= 3;
}

int main() 
{
    int num = 5;
    int* tmpPtr = &num;

    writeOutput(tmpPtr);
    *tmpPtr += 15; // add 15 to the data pointed to by tmpPtr
    writeOutput(tmpPtr);

    computeTriple(tmpPtr);
    writeOutput(tmpPtr);
    
    return 0;
}



Pointers to Functions


The pointer declaration must match the return data type and arguments to those of the function e.g.

Function looks like this:
int addArgs(int a, int b);
Therefore the function pointer will look like (the data types are matched):
int (*<function pointer name>) (int a, int b) = &<function name>;

Additionally, the pointer name must be enclosed within ( ) to avoid compilation errors.
int (*<function pointer name>)

#include <iostream>
using namespace std;

// inline function to output the total of 2 passed arguments
inline int addArgs(int a, int b)
{
    int total = a + b;
}

int main() 
{
    // create pointer pointing to addArgs function
    int (*functionPointer1) (int a, int b) = &addArgs;
    
    // use function - returns 12
    cout << addArgs(5, 7) << endl;
    // use pointer to use function - returns 26
    cout << functionPointer1(22, 4) << endl;

    
    return 0;
}
