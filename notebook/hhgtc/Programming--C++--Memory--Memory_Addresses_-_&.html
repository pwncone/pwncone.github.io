#Memory Addresses

-

&


Your programs store data in memory.
Memory is simply a big long list of addresses (in hexadecimal).
When you declare a variable, depending on the data type you specify, different sizes of memory will be reserved for your variable to be stored in e.g. the size of int is different to string etc.

You can reference/retrieve the memory address of where your variable is stored using &<variable>
#include <iostream>
#include <string>
using namespace std;

int main() {
    int num = 1000;
    string string1 = "much luls";

    cout << &num << endl;
    cout << &string1 << endl;
    
    return 0;
}
