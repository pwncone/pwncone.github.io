# Drawing


Drawing text & lines whilst internal.

## Use the game's graphics API


You could use whatever graphics API the game is using to draw your text/boxes
e.g. DirectX9, DirectX11, etc.
For example, CSGO uses DirectX9. Therefore, you use DirectX9 to draw your cheats.

## Use the game's internal functions


Instead of interacting with the game's graphics API, you could use the game's internal functions to draw inside.
The game's internal functions are most likely wrappers around the graphics API.

This is common in CoD cheats as they have a lot of internal functions to draw,
like drawing text and boxes.

It's also common in barebones/simple assembly cheats where you don't want to write your own drawing code,
so you just use the game's instead.

## Use an external graphics library


Use an external graphics library like ImGui or Nuklear to draw text/boxes.
This is getting pretty common nowadays.

## Create an Internal Overlay


Don't know much about this.
See this talked about with Battlefield 1/5 cheats instead of using ImGui.

## Hook a Trusted Overlay


Instead of drawing in Dx9's EndScene or Dx11's Present, you could hook a function in a trusted overlay and draw there: Discord, Steam, Origin, etc.
These modules often have DLLs loaded into the game so they can draw their overlays.
The DLLs are trusted because they're legitimate applications.
You can hook some of these overlays to draw your cheat overlay instead.
