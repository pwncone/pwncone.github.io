Use GetModuleFileName with the NULL parameter to get the full path of the current file name.
https://docs.microsoft.com/en-gb/windows/win32/api/libloaderapi/nf-libloaderapi-getmodulefilenamew

// get full path of current file name
wchar_t filename[MAX_PATH];
GetModuleFileNameW(NULL, filename, MAX_PATH);
printf("filename: %S \n", filename);
