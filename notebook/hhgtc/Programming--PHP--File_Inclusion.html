# File Inclusion


File Inclusion is where you include the content of one PHP file in another PHP file before the server executes it
e.g. include getstats.php into statsreport.php

Can use
• include()
• require()

## include()


include() takes all the text in the specified file and copies it into the file doing the include
If there's an error, include() will create a warning and continue executing

## require()


require() takes all the text in the specified file and copies it into the file doing the require
If there's an error, require() will give a fatal error and halt execution

RECOMMENDED TO USE require()
