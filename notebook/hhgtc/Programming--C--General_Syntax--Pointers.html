# Pointers


Pointers store memory addresses.
Accessing the value that the pointer points to (not the memory address) is called dereferencing.

&var - refer to memory address of variable
*var_pointer - refer to data at memory address <-- this is DEREFERENCING

int var = 1;                // initialise variable called 'var'
int *var_address = &var;    // intialise pointer to 'var'

// Use 'var_addrsess' to accesses the memory address - prints 0x7cffe...
printf("var_address = %p \n", var_address);

// Use '*var_address' to accesses the data at memory address - prints 1
printf("var_address value = %d \n", *var_address);


## Pointers and Strings


char *hey = "Hello"; will store the memory address of the first character of the string - H.
Memory is sequential, so C will continue to read the string until it reaches a NULL - \0 - byte
