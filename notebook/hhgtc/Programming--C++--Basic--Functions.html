#Functions


Functions are blocks of code that you can call later on to carry out a purpose/function for you

return_type function_name( parameter list ) {
   body of the function
   
   return <data>;
}


Return Type - the type of the value that the function returns
Function name - the actual name of the function
Paremeters - you pass parameters to a function. these parameters can these be a function

Example of a function that returns the max of 2 numbers:
int max(int num1, int num2) {
   // local variable declaration
   int result;
 
   if (num1 > num2)
      result = num1;
   else
      result = num2;
 
   return result; 
}

cout << max(5, 12) << endl;


To call a function
• you pass the required parameters along with function name
• if the function returns a value, you can store the returned value in a variable (or just use it)

In the above example, it's this:
max(5.12)
If there were no paremeters, you'd write this:
max()
