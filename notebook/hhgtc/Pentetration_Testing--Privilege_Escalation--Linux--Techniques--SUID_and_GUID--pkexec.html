# pkexec


• pkexec lets you execute a command as another user.
• I THINK it's normal for pkexec to have a SUID bit on it
◇ but if you have a password for the user, and see pkexec with a SUID bit on it, try pkexec anyway
▪ even if you can't run sudo on the machine!
◇ it might still work (as it does in the past for me)

Examples
• OSCP 10.11.1.219 - punchout

pkexec whoami
pkexec /bin/sh
