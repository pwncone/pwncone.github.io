# Format Strings


Both magical and dangerous.

## Format Data Type Speicifiers


Format specifiers specify the data type when you go to print a variable.

%d - int
• to ensure an integer fills 7 spaces, you can use %7d
• you can fill the blank spaces with 0s - %07d
%ld - long
◇ use for dword type
%f - float
• %.2f to show just 2 decimal
• floating point values are rounded if the format specifier allocates fewer decimal places (it's not simply truncated)

%c - char
%s - string
%S - wide character string

%hs - An ANSI string in an L"" wide character string
https://docs.microsoft.com/en-gb/windows/win32/api/winuser/nf-winuser-wsprintfa?redirectedfrom=MSDN

%p - pointer / memory address

## Specify Length


- - justify right
%-s
%-18s - justify right, pad until 18 length

. - truncate
%.18s - truncate to 18 characters

Truncate to length using variable:
char text[] = "hello there!";
int len = 5;
printf("* encrypted string: %.*s \n", len, text);

Need to truncate some lines and pad others?
Have to perform length checking
// for neat output -> truncate process name if over max length, pad if not
if (_tcslen((process_info + i)->pProcessName) > 24)
{
    _tprintf(TEXT("%.24s"), (process_info + i)->pProcessName);
}
else
{
    _tprintf(TEXT("%-24s"), (process_info + i)->pProcessName);
}
