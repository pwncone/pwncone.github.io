# Uploading Files


FOR YOUR BOUNDARIES TO BE READ CORRECTLY, (i.e. for the file to not be merged):
the boundary used in the form data around the uploaded file must be 2 - longer than the boundary specified in the Content-Type header:
Explained in first paragraph of first answer here: https://stackoverflow.com/questions/47878566/wininet-upload-file

e.g:
• Content-Type: multipart/form-data; boundary=----9132\r\n <- HAS 4 -

• ------9132\r\n <- HAS 6 -
Content-Disposition: form-data; name="upload"\r\n
------9132--\r\n <- HAS 6 -
