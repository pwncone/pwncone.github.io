# Nibbleblog


Nibbleblog is a “blog system” written by Diego Najar.
It's vulnerable.

## URLs


• /README will identify the Nibbleblog version
• /admin.php is the admin panel login page.
• /admin, /content, /languages, /plugins, /themes are all (potentially open) directory listings, which you can browse.
◇ If enabled, might be able to find username in /content/private/users.xml

## Tips


### Gobuster


If you gobuster the site, brute-force for .php and .txt extensions as well.
You'll get more results.
gobuster -e -u http://10.10.10.75/nibbleblog/ -w /usr/share/seclists/Discovery/Web-Content/big.txt -x .php,.txt
