# main & WinMain & DllMain


These define the program entry point.

Use main for console applications.
◇ main - ANSI cmdline args
◇ wmain - Unicode cmdline args

Use WinMain is for GUI applications (no console created for it at startup)
◇ WinMain - ANSI
◇ wWinMain - Unicode

Use DllMain for DLLs

## wWinMain


int WINAPI wWinMain(HINSTANCE hInstance, HINSTANCE hPrevInstance, PWSTR pCmdLine, int nCmdShow);

WINAPI is the calling convention.
It defines the order that parametres appear on the stack.

hInstance - handle to identify the .exe in memory
hPrevInstance - used in 16bit Windows. Now deprecated. Always 0.
pCmdLine - pointer to cmdline args
nCmdShow - flag defining if application window starts minised, maximised, or normal
