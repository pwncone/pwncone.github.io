# Structs - C vs C++


In C++, you can simply do this:
struct Player {
    char unknown1[4];
    float x;
    float y;
    float z;
    char unknown2[0x30];
    float yaw;
    float pitch;
};

Player player = NULL;


However, in C the Player data type will be "undefined".
Instead, in C you have to typedef the struct.
This solution is commonly found in Windows APIs e.g.
https://docs.microsoft.com/en-us/windows/win32/api/winternl/ns-winternl-peb
typedef struct _Player {
    char unknown1[4];
    float x;
    float y;
    float z;
    char unknown2[0x30];
    float yaw;
    float pitch;
} Player;

Player player = NULL;
Player* p_player = NULL;
