# Assembly


Assembly is a low level programming langauge which has a close relationship to the processor's machine code instructions.
These instructions can be statements like mov, add, push, and pop.

Different processors have different instruction sets, and therefore have a different assembly language.
Examples include:
• x86 - used on 32bit Intel and AMD processors
• x86-64 - an extension to x86 that's used on 64bit Intel and AMD processors
• ARM - a RISC (Reduced Instruction Set) typically found in mobile processors
• MIPS - mostly found in routers and gateways - the MIPS microprocessor
• PowerPC - can be found in the Nintendo Gamecube and older Apple Macs

Assembly code gets converted into executable machine code by an assemebler.
