# GDI


Very simple and easy to use.
All windows use GDI as a base, so you don't have to do anything extra - just draw.
GDI+ is an extension of GDI.
Despite simplicity, has issues like flickering.

Use GetDC to grab the device context of the window.
Then use FillRect to draw.

Refer to the How to make a GDI ESP - Graphics Device Interface - GDI32.dll heading:
https://guidedhacking.com/threads/how-to-make-an-esp-hack-for-any-game.8313/
