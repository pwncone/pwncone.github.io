The idea of a file upload is:
• user uploads a file to your scripts
• that file gets stored in a temporary location (like /tmp) with a tmp name
• your script checks the file is good
• if good, your script moves the file to permanent storage with move_uploaded_file()
• once your script has finsihed, the temp file will atuomatically be deleted

$_FILES is an associative double dimension array.
It stores all the info related to the uploaded file
• $_FILES['file']['tmp_name'] − the uploaded file in the temporary directory on the web server
• $_FILES['file']['name'] − the actual name of the uploaded file
• $_FILES['file']['size'] − the size in bytes of the uploaded file
• $_FILES['file']['type'] − the MIME type of the uploaded file
• $_FILES['file']['error'] − the error code associated with this file upload
