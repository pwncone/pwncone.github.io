# Tornado


Tornado is a scalable, non-blocking (asynchronous, it can handle a large number of concurrent connections) web server and web application framework written in Python. It was developed for use by FriendFeed; the company was acquired by Facebook in 2009.

Modules
• An asynchronous MongoDB driver called Motor.
• CouchDB drivers called corduroy and trombi.
• Asynchronous driver for PostgreSQL wrapping psycopg called Momoko
