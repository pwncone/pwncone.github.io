## Architecture


https://docs.microsoft.com/en-us/windows/win32/direct3d9/direct3d-architecture

D3D Device
This is the graphics adapter (physical GPU card, virtual device, etc).
From the device, you create graphics objects: Swap Chain, Textures, Shaders, Buffers, etc.

From the Swap Chain, Textures, etc.
you can create further objects

Swap Chain
Is how the pixel data gets to the monitor

Device Conext is used for drawing.
Immediate context - gets executed immediately on the 3d driver
Deferred context - build up a queue of commands and then send to immediate context
