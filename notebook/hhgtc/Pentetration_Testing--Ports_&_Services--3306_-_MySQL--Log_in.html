## Log In


Always test:
username: root
password: root

Different ways of logging in:
mysql -u root
mysql -u root -p
mysql -h <target ip> -u root
mysql -h <target ip> -u root -p
mysql -h <target ip> -u root@localhost
mysql -h <target ip> -u ""@localhost

telnet 192.168.0.1 3306

You'll probably get this message a lot:
ERROR 1130 (HY000): Host '192.168.0.101' is not allowed to connect
happens because root is configured so it can only login from 127.0.0.1/localhost
