#Compiling


To execute a C++ program, you first have to compile it.
A compiler reads the source code of your program and translates it into machine code.

Linux


On linux, you can use the GNU C++ compiler - g++
Install it if you don't have it (you should) - apt install g++

Compile:
g++ file.cpp -o file

Run your program:
./file

Windows


The GNU project was launched in 1984 to develop a free (as in freedom) Unix-like OS
Part of GNU is Minimalist GNU for Windows (MinGW)
MinGW includes the GNU C++ compiiler that can be used on Windows to compile executables

1) Install MinGW using the gui setup:
http://www.mingw.org/wiki/Getting_Started
Choose the base bin and the g++ compiler

2) Add MinGW \bin directory to your PATH variable
start > search ‘variable’ > pick ‘edit system variables’ > find Path > edit it to include MinGW \bin folder like this: C:\MinGW\bin; (the semicolon is important)

3) Check install
Check if it's installed by opening cmd and typing g++ -v (you should get a version number printed out)

Compile:
g++ file.cpp -o file.exe

Run your program:
file.exe
