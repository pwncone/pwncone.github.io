# Apache Web Server


Apache is probably the most common web server you'll find.
This is a collection of general notes about Apache.

## Apache Shellshock


If the Linux host on which Apache is running is vulnerable to Shellshock, you might be able to exploit Shellshock by appending code to the end of the Apache HTTP headers.

Refer to the Vulnerabilities > Linux > Shellshock for more info.

## CGI Exploits


Apache web servers can run CGI scripts.
CGI scripts are exploitable.

Refer to the Vulnerabilities > Web > CGI Scripts for more info.
