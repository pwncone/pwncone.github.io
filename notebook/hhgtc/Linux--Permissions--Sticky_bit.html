# Linux - Permissions - Sticky bit


• A sticky bit means that only the owner of the file (or root) can delete or modify the file.
• The sticky bit is very useful for shared directories or files
◇ where multiple users have write access but you don't want them to be able to delete it

chmod +t <file>
chmod 1755 <file>

The T denotes the sticky bit
-rw-r--r-T 1 jane jane 18 Feb 24 20:46 sticky.txt
