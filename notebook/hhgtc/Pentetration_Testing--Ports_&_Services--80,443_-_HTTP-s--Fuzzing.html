# Fuzzing


The goal of fuzzing within wider hacking is to find vulnerabilities.
In the context of website, fuzzing gives you more fine-grained control over what data you send to the web server.
e.g. instead of just bruteforcing for directories, you can apply a mutation to a specific bruteforce
e.g. bruteforce for directoryN where N is a number increasing from 0 - 100

## patator


• Might return UnicodeDecodeError if trying to decode an image

patator http_fuzz url=http://10.0.0.1/FILE0 0=/usr/share/seclists/Discovery/Web-Content/big.txt -x ignore:sclen=<char length>
