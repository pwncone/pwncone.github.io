# Linux - Commands - Editing files/text



## sed


sed - basic editing of streams i.e. text
sed -i 's,foo,bar,g' file.txt - Replace instances of foo with bar in file.txt
sed "1d" - remove first line of stream/text

## cut


cut - cut out text by ‘column’/delimteter
-d defines the delimiter
-fN - defines the field to cut
cut -d “ ” -f 4 file.txt - split up the file into fields using “ ” (spaces) and cut out the 4th character in each field from file.txt

## tr


tr is used to translate A into B from a file.
tr "[:lower:]" "[:upper:]" < lower.txt > higher.txt - translate all lowercase letters into uppercase letters
cat file.txt | tr "." "_" - translate all . into _
cat file.txt | tr -d "=" - remove all = from file.txt
echo "mVGZ3O3omkJLmy2pcuTq" | tr 'A-Za-z' 'N-ZA-Mn-za-m' > norot13.txt - rot13 using tr

## awk


awk is advanced. It iterates over the whole file line by line.

Basic structure of awk command:
awk '/search_pattern/ {action_to_take_on_matches; another_action;}' file_to_parse

awk ‘{print}’ <file> - print out every line of the file
awk ‘/192.168.1.100/' error.log - filter out a specific IP (will print that line)

## nano


A user-friendly terminal file editor.
ctrl+w to save and quit

nano <file>

## vim


A less user-friendly text editor. If nano isn't available, vim most likely will be.
Vim stand for Vi improved.

vim <file>
:wq - to write file and quit

## vi


An older version of vim.
vi <file>
