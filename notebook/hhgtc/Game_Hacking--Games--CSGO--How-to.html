## Bones


WARNING:
The bones are different for each model in the game.
The only bone that stays consistent is 8 - the head bone.
If you're creating a bone ESP, the only solution is to loop through entire list of bones
and draw a line between every bone.

Bone IDs:
â€¢ 8 - head bone (stays the same on every character model)

/*
Each bone is 0x30 bytes long 
and there's 0x10 bytes of padding between each of the coordinates for the bone (starting at 0x0c).
*/
vec3 GetBoneCoordinates(Entity* entity, int bone_id)
{
uintptr_t bone_base = entity->bone_matrix;
vec3 bone_position = { 0 };
bone_position.x = *(float*)(bone_base + 0x30 * bone_id + 0x0c);
bone_position.y = *(float*)(bone_base + 0x30 * bone_id + 0x1c);
bone_position.z = *(float*)(bone_base + 0x30 * bone_id + 0x2c);

return bone_position;
}
