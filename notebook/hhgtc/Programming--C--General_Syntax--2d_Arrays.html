# 2d Arrays


2d arrays have rows (left to right) and columns (up and down).



## Declare & initialise a 2d array


// int md_array[rows][columns]
int number_array[3][4] = {
                            {0, 1, 2, 3},       /* populate/initialise row 0 */
                            {4, 5, 6, 7},       /* populate/initialise row 1 */
                            {8, 9, 10, 11}      /* populate/initialise row 2 */
};


## Accessing elements in a 2d array


// Access row 2, column 3 = 11   (arrays start at 0)
int value = number_array[2][3];
