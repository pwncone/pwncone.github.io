#Variables


Variables are like containers that you store data in. You can refer to the variable every time you'd like to use its data

Declaring variables


Declare variables with data-type variable;

To declare the variable foo as an integer, we would to use the following syntax:
int foo;

To declare and initalise the variable bar as an integer, we would do:
int bar = 1;

Multiple variables of the same data type can be declared/intialised on the line
int a = 0, b = 1, c = 2, d = 3, e = 4;


Declaring vs Initialisation
Declaring a variable:
int a;
The variable a can be used, but since we didn't initialise it, it doesn't have any data in it

Initialising a variable is assigning it a value after it's been declared e.g.
int a = 5;

Strings


You have to use the string library #include <string>
#include <iostream>
#include <string>
using namespace std;

int main() {
    string phrase = "Elephant Academy";
    cout << phrase << "\n";

    return 0;
}



Local Variables


Variables that are declared inside a function are local variables.
They can be used only by statements that are inside that function or block of code e.g.
#include <iostream>
using namespace std;
 
int main () {
   // Local variable declaration:
   int a, b;
   int c;
 
   // Initialization
   a = 10;
   b = 20;
   c = a + b;
 
   cout << c;
 
   return 0;
}



Global Variables


Global variables are defined outside of all the functions, usually on top of the program.
A global variable can be accessed by any function. That is, a global variable is available for use throughout your entire program.
#include <iostream>
using namespace std;
 
// Global variable declaration:
int g;
 
int main () {
   // Local variable declaration:
   int a, b;
 
   // Initialization
   a = 10;
   b = 20;
   g = a + b;
  
   cout << g;
 
   return 0;
}


If a global variable and local variable use the same name, the local variable will take preference over the global inside of the function e.g.
#include <iostream>
using namespace std;
 
// Global variable declaration:
int g = 20;
 
int main () {
   // Local variable declaration:
   int g = 10;
 
   cout << g;
 
   return 0;
}
