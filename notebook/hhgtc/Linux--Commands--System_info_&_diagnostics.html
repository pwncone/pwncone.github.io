# Linux - Commands - System info



date - print or set system date
cal - show calendar
uptime - how long the system has been running
w - show who is logged on and what they're doing
finger <user> - get info on a user
hostname - show hostname of system
hostnamectl set-hostname <name> - change hostname of system
uname -a - show kernel/system info
cat /proc/cpuinfo - show cpu info
cat /proc/meminfo - print memory info
free -m - show how much memory is currently in use
vmstat 1 20 - report virtual memory statistics
iostat - mointor disk usage and I/O

lsusb - list USB devices
lspci - list PCI devices
lsscsi - list SCSI devices

lsof - list open files and their assosciated processes
fuser - shows information about the process that is using the file
ps m - view process threads
