# CMS Made Simple


CMS Made Simple is an open source content management system to provide developers, programmers and site owners a web-based development and administration area. It's basically another admin panel.

## Locations


• /admin - Login page
• /config.php - Database credentials are stored here

## Database Structure


https://cmscanbesimple.org/blog/cms-made-simple-admin-password-recovery
https://forum.cmsmadesimple.org/viewtopic.php?t=14467

You can read usernames from the cms_users table.
select user_id,username,password from cms_users;
