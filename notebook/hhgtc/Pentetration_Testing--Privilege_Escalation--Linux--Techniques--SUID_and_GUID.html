SUID binaries run with the permissions of the user who owns it.
GUID binaries run with the permissions of the group who owns it.

There's lots of different ways to find SUID and GUID binaries.
Some produce results, some don't. I have no idea why. It seems like everybody uses a different method.
Here's them all.

2000 = GUID
4000 = SUID

Files
find / -uid 0 -perm -4000 -type f 2>/dev/null
find / -perm -4000 -type f -exec ls -la {} 2>/dev/null \;

Files & Directories
find / -perm -u=s 2>/dev/null
find / -perm -g=s 2>/dev/null
find / -perm 4000 2>/dev/null

Directories
find / -perm -u=s -type d 2>/dev/null
find / -perm -g=s -type d 2>/dev/null
