# Port Forwarding - proxychains



## Use proxychains-ng!


https://github.com/rofl0r/proxychains-ng

• Old proxychains causes seg faults with Nmap
◇ it also isn't maintained anymore
• Installation instructions are on github

• proxychains doesn't support UDP or ICMP

## Install


git clone https://github.com/rofl0r/proxychains-ng.git
./configure --prefix=/usr --sysconfdir=/etc
make
sudo make install
sudo make install-config (installs proxychains.conf)

## Usage


-q for quiet output

proxychains4 <command>

## Troubleshooting


• Disable DNS through socks
◇ in /etc/proxychains.conf file, just add a # before the line "proxy_dns"
