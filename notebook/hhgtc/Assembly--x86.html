# x86 Assembly


The x86 architecture was developed by Intel.
Over time it has expanded, and an extended instruction set was released to support 64bit processors called x86-64.
Therefore, x86 typically refers to the entire x86 family of microprocessors.

• IA-32 - Intel Architecture 32-bit - refers to the 32bit version of the x86 instruction set architecture.
• IA-64 - Intel Architecture 64-bit - refers to the 64bit version of the x86-64 instruction set architecture.
