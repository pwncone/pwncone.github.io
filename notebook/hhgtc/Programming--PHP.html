# PHP


• by Rasmus Lerdorf
• intitial relase: 1994
• recursive acronym for "PHP: Hypertext Preprocessor"
• It's a server side scripting language that is embedded into HTML
• integrates well with datbases
• pleasantly fast
• designed to be as forgiving as possible
• syntax is C-like
• php can:
◇ perform system functions
◇ add/remove/modify databases
◇ access & set cookies
◇ handle POSTed data
◇ etc.

Introduction


• PHP code gets stripped from the HTML output (so end users can't see it)
• All PHP code must be included using 1 of these 3 markup tags recognised by the PHP parser
◇
<?php PHP code goes here ?>
<?    PHP code goes here ?>
<script language = "php"> PHP code goes here </script>


Environment


• PHP requires 3 components:
◇ a web server
◇ a database
◇ a PHP parser
▪ parses/executes the PHP and generates HTML output to send to browser
