# History


https://www.computerhope.com/history/unix.htm

• In 1964, Bell Labs make the Multics operating system
• In 1969, Ken Thompson and Dennis Ritchie of Bell Laboratories developed the UNIX operating system
◇ Unix was originally wirtten in Assembly
• In 1973, Version 4 Unix was rewritten in C
• In 1975, Unix v6 was released as a distribution, including the full source code
◇ Unix v6 was the first version of the Unix operating system to see wide release outside Bell Labs
• In the 1980s, companies begin to develop their own flavours of Unix
◇ AT&T makes System V unix
◇ Berkley makes UnixBSD (berkeley software distribution)
◇ Sun Microsystems makes Solaris
◇ Microsoft + Santa Cruz Operation make XENIX
◇ DEC makes ULTRIX
• In 1983, Richard Stallman announced the GNU (GNU is Not UNIX) project
◇ GNU's goal is to provide a distributable replacement for UNIX
• In 1992, Linus Torvalds introduced the Linux kernel
◇ This is when the goal of the GNU project was achieved, as Linux was released under the GNU GPL - GNU General Public License
