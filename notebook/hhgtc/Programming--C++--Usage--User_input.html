#User input


User input is taken using the <iostream> standard library, the cin method takes user input
You save it into a variable (declare the variable first)

Taking numbers
Use cin
#include <iostream>
using namespace std;

int main() {
    int age;
    cout << "Enter your age: ";
    // take user input, store in age variable
    cin >> age;

    cout << "You are " << age << " years old!" << endl;

    return 0;
}


Taking strings
When taking strings, use getline() function (because cin >> will only take the first word, ignoring whitespace)
getline(cin, <variable>);
#include <iostream>
#include <string>
using namespace std;

int main() {
    cout << “What's your name? ”;
    string name;
    getline(cin, name);
    
    cout << “Your name is ” << name << endl;

    return 0;
}
