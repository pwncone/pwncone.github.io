# Port 110 - POP3 Unencrypted


POP3 - Post Office Protocol - is used for retrieving emails from a mail server.

POP3
Connects to email server at intervals to a download all of a user's emails without leaving copies on the server

IMAP
Stays continuosly connected to the server and mirrors the mailboxes on the server to the mailboxes of the client

POP3 is older and was designed for when users only accessed email from one device (the emails get downloaded to that device and then deleted from the server. means you would get different emails on computer, phone, tablet etc.)

By default, POP3 run on two ports:
• Port 110 - unencrypted
• Port 995 - SSL-POP port (encrypted)

## Checklist


• Connect & grab banner
◇ nc 10.11.0.4 110 -v
 ◇ telnet 10.10.10.120 110
 ◇ openssl s_client -connect <IP>:995 -crlf -quiet
• Check version for exploits
• Run nmap scripts
◇ nmap -p 100,995 -sV --scripts "pop3-capabilities or pop3-ntlm-info" 10.11.0.4
 ◇ nmap -p 100,995 -sV --scripts "pop3*" 10.11.0.4
• Bruteforce

# Log in
USER myusername
PASS mysecretpassword

# STAT - show number of messages
STAT displays the number of messages currently in the mailbox and their size in bytes
STAT

# LIST - list the messages in the mailbox
# LIST lists the messages in the mailbox with an ID and their size in bytes
LIST

# RETR - retrieve messages
RETR <message id>
RETR 1

# TOP - retrieve part of a message
# Return headers only:
TOP 1 0
# Return headers and first 10 lines of body:
TOP 1 10
