# Password Cracking - .7z (7zip)


7zip files require a bit more work to crack.

## Extract hash from zip.7z


First, extract the hash from the 7z file using 7z2hashcat. - https://github.com/philsmd/7z2hashcat.git
I had to install libcompress-raw-lzma-perl to get it to work.
The extracted hash in the code box below has been shortened (they can be incredibly long).

git clone https://github.com/philsmd/7z2hashcat.git
apt install libcompress-raw-lzma-perl
./7z2hashcat.pl backup.7z
