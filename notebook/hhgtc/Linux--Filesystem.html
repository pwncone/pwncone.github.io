# Linux - Filesystem


/bin - command line utilities
/boot - kernel boot loader files
/dev - device files (think drivers)
/etc - config files (user list, password list, software configs)
/home - personal directories for each user + their documents
/lib - library files that binaries can use
/media - the mount point for removeable media - dvds, usb sticks etc.
/mnt - temporarily mounted filesystems
/net - mount point for network directories
/opt - optional application software packages
/proc - currently running processes
/root - root's home directory
/sbin - contains essential system binaries, usually can only be ran by root
/srv - site specific data served by the system
/tmp - temporary files (programs just dump shit here, you can too)
/usr - programs/libraries/docs for user-installed programs
/var - used for variable data that changes often e.g. log files, user tracking, caches etc.
