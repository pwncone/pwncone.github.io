# Nostromo


https://www.gsp.com/cgi-bin/man.cgi?section=8&topic=nhttpd

Nostromo is an open-source web server designed by Marcus Glocker.
It runs as a single process, and handles normal concurrent connections by select calls.
It's written in C.

## HOMEDIRS


The Nostromo man page describes what HOMEDIRS are - http://www.nazgul.ch/dev/nostromo_man.html

The homedirs option lets you serve the home directories of users via HTTP.
The URL of the user's home directory is - http://example.com/~user/

The homedirs_public specifies specific sub-directories in a users' home folder to be included (instead of the entire home folder). An example directory might be public_www

In /home is david, which we can't read.
Have a look in david's /home/david/public_www folder -> you might be able to read it.
Folders you find in here you can then browse to on the website:
http://10.10.10.165/~david/protected-file-area
