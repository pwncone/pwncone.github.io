- Stringising Operator


The preprocessor # operator is known as the ‘stringising’ operator.
It will print everything apart from whitespace at the end of the file
#define ONELINE(str) cout << #str << endl

A macro definition can combine 2 strings into a single string using ##
#define COMBINELINES(a, b) cout << a##b << endl

#define ONELINE(str) cout << #str << endl
#define COMBINELINES(a, b) cout << a##b << endl
