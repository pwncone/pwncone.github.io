# Python Libraries


If you can write to a python library (which you normally can't do),
as long as there's a script running as root that imports said writable library, you can get code execution.

When python libraries are imported,
e.g. import os,
all the code in that library gets executed.
If the script importing the library is running as root, all code in the os library will get executed as root.

Python library location - /usr/lib/python2.7/
Python syntax to execute system commands
import os
os.system("touch /tmp/file.txt")

Examples are in sub-nodes.
