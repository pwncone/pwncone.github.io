# Internal


Internal cheats run inside of the game's process.
Typically this is done by injecting a DLL into the game's memory.
From there, you can read from and write to values within the game without using API calls.

Performance is fast.
However, depending on the game/anticheat, evading detection has a lot of parts to it.
