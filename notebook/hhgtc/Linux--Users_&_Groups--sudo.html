# Linux - Users - sudo


• Whilst root can do anything on the system, it's generally advised not to use root when performing administrative tasks
◇ This is because you can easily mess something up
• As a safeguard against the powers of root, a user can run a single command as root with sudo
◇ sudo stands for superuser do
• sudo allows users to run commands as root

◇ To give a user sudo permissions, they must be added to the /etc/sudoers file
a) use visudo is used to edit the /etc/sudoers file
b) add the user to the sudo group - usermod -a -G sudo <user>

sudo -l - list the commands you can run with sudo
