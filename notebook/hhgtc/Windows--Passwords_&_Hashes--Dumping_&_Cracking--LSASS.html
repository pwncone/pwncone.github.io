# Dumping & Cracking - LSASS


Passwords dumped from LSASS will be in plaintext, so there's no need to crack them.

## Mimikatz


Mimikatz can dump from LSASS memory.
Tools > Mimikatz

You will have to drop the mimikatz binary to disk.

## Invoke-Mimikatz


If don't want to drop a binary to disk, you can load Mimikatz into memory via PowerShell
Invoke-Mimikatz
