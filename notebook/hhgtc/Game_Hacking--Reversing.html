### Suspend and Dump


If the game is protected / can't attach to debugger / might get banned for dumping,
try suspending the process first, then attaching x64 dbg, and dump with Scylla.

DON'T RESUME THE PROCESS AFTERWARDS.
You can detect whether your process has been suspended using GetTickCount.

You can use Process Hacker to suspend the process,
but some games don't like Process Hacker.
An alternative is resmon - a built in Windows utility. Suspsend from there instead.
