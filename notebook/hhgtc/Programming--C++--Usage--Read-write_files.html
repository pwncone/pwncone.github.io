#Read/write files


The standard C++ library <fstream> provides functions for working with files.
#include <fstream>


#Opening files for read/write


For each file opened, you have to create a filestream object.
There are 2 ways to do this:

Option 1)
For writing data to a file- use ofstream (output file stream)
ofstream writeFile("writeStory.txt");

For reading data from a file - use ifstream (input file stream)
ifstream readFile("readStory.txt");


Option 2)
Use fstream and specify the ios::openmode (imo, the preferred method)
Write:
fstream writeFile("writeStory.txt", ios::app);

Read:
fstream readFile("readStory.txt", ios::in);

Both read and write:
fstream readWriteFile("story.txt", ios::in | ios::out);


ios::openmode


When you open a file, you can specify the file's “mode” to control the behaviour of that filestream object
i.e. you can choose to append data, truncate the entire file, open it exclusively for reading etc.

fstream story("story.txt", ios::app);


ios::out - Open a file for writing.
ios::in - Open a file for reading.
ios::app - Append mode. Write data to the end of the file (append data)
ios::trunc - If the file already exists, its contents will be truncated before opening the file (default behaviour)
ios::ate - Open a file without truncating and allow data to be written anyhwere in the file
ios::binary - Treat the file as binary format rather than text, so the data may be stored in non-text format.

You can specify multiple modes using | (a pipe)
e.g. to open a binary file for writing to
fstream binaryFile("binary.bin", ios::out | ios::binary);



#Closing a File


When a C++ program terminates <fstream> automatically flushes all the streams, release all the allocated memory and close all the opened files.
BUT it's always good practice to close open files before the program ends anyway

fstream story("story.txt", ios::app);
//code goes here
story.close();



#Reading files


Make a variable to store what you're reading into.
use .get() to get/read individual characters
use getline() to get/read whole lines

Loop through a file per character
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    char fileCharacter;
    fstream readFile("poem.txt", ios::in);
    for(int i = 0; !readFile.eof(); i++) {
        readFile.get(fileCharacter);
        cout << fileCharacter << " ";
    }

    readFile.close();
    return 0;
}


Loop through a file per line
#include <iostream>
#include <string>
#include <fstream>
using namespace std;

int main() {
    string fileLine;
    fstream readFile("poem.txt", ios::in);
    for(int i = 0; !readFile.eof(); i++) {
        getline(readFile, fileLine);
        cout << "Line " << i << ": " << fileLine << endl;
    }

    readFile.close();
    return 0;
}


Loop through a file per line - while loop instead
#include <iostream>
#include <string>
#include <fstream>
using namespace std;

int main() {
    fstream readFile("testfile.txt", ios::in);

    string fileLine;
    while(!readFile.eof()) {
        getline(readFile, fileLine);
        cout << fileLine << endl;
    }
    
    return 0;
}



#Writing files


Create the filstream object, and write your text to the file
fstream story("story.txt", ios::app);
story << "Chapter 2" << endl;


Error handling


Before writing to a file, it's a good idea test whether the filestream object has actually been created.
Usually you do this with an if statement
fstream poem("poem.txt", ios::app);
if(!poem) {
    cout << "Error opening file for output" << endl;
    return -1;
}
poem << "Hey there" << endl;
poem.close()
