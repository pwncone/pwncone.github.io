## Installation


git clone https://github.com/Screetsec/TheFatRat.git
cd TheFatRat
chmod +x setup.sh && ./setup.sh


Recommend letting fatrat add itself to PATH.
Then can run with fatrat

### Update


cd TheFatRat
./update && chmod +x setup.sh && ./setup.sh


### Troubleshoot fatrat


chk_tools script to use in case of problems in setup.sh of fatrat
This script will check if everything is in the right version to run fatrat and will also provide you a solution for the problem.
cd TheFatRat
chmod +x chk_tools 
./chk_tools
