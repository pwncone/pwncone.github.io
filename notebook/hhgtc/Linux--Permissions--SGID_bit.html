# Linux - Permissions - Set Group ID


Similar to the set user ID permission bit, there is a set group ID (SGID) permission bit.
This bit allows a program to run as if it was a member of that group.

Add a SUID bit to a file:
chmod g+s <file>
chmod 2555 <file> - the numerical representation for SGID is 2

The S denotes the GUID bit
The binary/file might also be highlighted
-rwxr-Sr-x 1 bob bob 18 Feb 24 20:46 askMike
