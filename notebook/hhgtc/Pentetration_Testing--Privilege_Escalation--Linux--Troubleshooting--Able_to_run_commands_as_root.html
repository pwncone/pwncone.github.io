# Able to run commands as root?


You have options.

## Reverse Shell


Just run a reverse shell back to your attacking system to get a root shell.

## Add user to sudo group


You could add yourself (the current user you have control of) to the sudo group.
You'll have to know the user's password.
usermod -aG sudo <username>

## Read /etc/shadow


You can grab /etc/shadow and attempt to crack the passwords.
cat /etc/shadow
