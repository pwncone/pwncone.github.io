#Data types


When you declare a variable, you reserve space for it in memory.
The data type is what defines how much space in memory is reserved.
Choose the data type appropriate to what will be returned by your function e.g. your function returns a number, use int

Basic data types:
char ‘A’ A single byte, capable of holding one character (char must always single quotes, not double quotes)
int 100 An integer (a whole number)
float 0.123456 A floating point number to 6 decimal places
double 0.1234567890 A floating point number to 10 decimal places
bool true (1) or false (0) A Boolean value of true (1) or false (0)

Extra:
void Valueless
wchar_t Wide character

The size of these data types can vary depending on the compiler/computer

Type Modifiers


The data types can be modified above using the following:

signed positive or negative numbers
unsigned positive numbers (i.e. no signs e.g. - or .)
short short
long long
