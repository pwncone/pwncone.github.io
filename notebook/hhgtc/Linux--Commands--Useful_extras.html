# Linux - Commands - Useful extra commands


A list of generally useful stuff that I can't fit anywhere else

## watch


watch a file for changes
watch -n 5 cat file.txt - run cat file.txt every 5 seconds and watch for changes

## xxd


make or reverse a hexdump
Make a hexdump of a file: xxd <file> <save as>
Reverse a hexdump into a file: xxd -r <hexdump> <save as>

## mount a .vhd file on linux


https://xo.tc/how-to-mount-a-vhd-file-on-linux.html
install libguestfs-tools - apt install libguestfs-tools
make mount point - mkdir /mnt/vhd1
mount using guestmount - guestmount --add harddisk.vhd --ro /mnt/vhd1/ -m /dev/sda1
• --add option is for the image you want to access
• --ro sets to read only, alternatively you could use --rw for read / write
• /mnt/vhd the path where you want to mount the drive
• -m /dev/sda1 specify which partition within the .vhd file you want to mount
