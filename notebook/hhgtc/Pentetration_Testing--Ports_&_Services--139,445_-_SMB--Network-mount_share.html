## Network-mount share


You don't have to do this, it's just an option.

Install cifs utils
apt install cifs-utils
Create a mount point
mkdir /mnt/bastionBackups
Mount the volume
mount -t cifs //10.10.10.134/<share> <mount to>
• example #1 - mount -t cifs //10.10.10.134/Backups /mnt/bastionBackups
• example #2 - mount -t cifs -o user=bob,sec=ntlm,dir_mode=0077,vers=1.0 "//10.11.1.136/Bob Share" /mnt/bobMount
