# Linux - Vi/Vim


Vi/Vim is a text editor for winners/real linux users (omegalul).
Vim stands for vi (Improved). Vi is old, Vim is the new shit.

There are 2 modes in Vim:
• Command mode
◇ where you execute commands/navigate the file
◇ this is the mode you'll be in when you first open a file
• Insert mode
◇ where you insert text/edit the file

## Command Mode


### Searching a File


• / to search through the file fowards - e.g. /hey
• ? to search through the file backwards - e.g. ?hey

• n to go forward in results
• N to go backwards

### Navigation


There's no mouse because you're at a command line.
To move the cursour:
• h left
• j up
• k down
• l right
• Or the arrow keys :)

### Editing text


• x - used to cut the selected text
◇ also used for deleting characters
• dd - used to delete the current line
• y - ‘yank’ or copy whatever is selected
• yy - ;yank' or copy the current line
• p - paste the copied text before the cursor

### Saving and Exiting


• :w - writes or saves the file
• :q - quit out of vim
• :wq - write and quit
• :q! - quit out of vim without saving the file
• ZZ - equivalent of :wq, but one character faster

• u - undo your last action
• Ctrl-r - redo your last action

## Insert Mode


Press i to enter insert mode.

You should now see ‘INSERT’ at the bottom of the shell, indicating you're in command mode
To exit insert mode (and return to command mode) press ESC

There's also other ‘insert’ modes:
• O - insert text on the previous line
• o - insert text on the next line
• a - append text after the cursor
• A - append text at the end of the line
