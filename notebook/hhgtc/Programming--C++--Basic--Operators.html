#Operators


Operators are how you manipulate things (like maths)

Arithmetic operators


+ Add
- Subtract
* Multiply
/ Divide
% Modulus
++ Increment
-- Decrement

Comparison operators


== Check if equal
!= Not equal
> Greater than
< Less than
>= Greater than or equal to
<= Less than or equal to

Logical operators


&& Logical AND
|| Logical OR
! Logical NOT

Logical AND will return true if both of the compared operands are true
Logical OR will return true if one of the compared operands is true
Logical NOT will switch the boolean value of the operand e.g. a = true, !a, a now equals false

The term “boolean” refers to a system of thought developed by the English mathmetician George Boole (1815-1864)

Ternary operator


?
This operator first evaluates an expression for a true or false condition, then returns one of two specified values depending on the result of the evaluation

(test-expression) ? if-true-return-this : if-false-return-this ;

int a = 2;
int b = 5;
int c = 4
int d = (a == 2) ? c:b;

This will check if a is equal to 2.
Because a == 2, c gets returned.
If a didn't equal 2, b would get returned.

A common use of the ternary operator, using the % modulus operator, is to determine whether a value is odd or even
(var %2 != 0) ? if-true(odd)-do-this : if-false(even)-do-this ;
