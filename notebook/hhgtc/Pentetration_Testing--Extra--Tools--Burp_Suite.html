# Burp Suite


Notes on Burp Suite.

## Shortcuts


ctrl+U = URL encode highlighted string (very useful)
ctrl+shift+U = URL decode

## Set up a Burp Proxy


1. Proxy tab > Options > Proxy Listeners > Add
2. Bind to port: 8500 (you can pick any port you want)
3. Request handling:
Redirect to host: <host you want to connect> e.g. 10.10.10.14
Redirect to port: <port you want to connect to> e.g. 8080

Now whenever you connect to localhost:8500 it will redirect you to 10.10.10.14:8080

## Error code: SSL_ERROR_RX_RECORD_TOO_LONG


Downgrade to java version 10
root@kali:~/ctf/chaos# sudo update-alternatives --config java

## Error: unrecognized_name


The "unrecognized_name" error indicates that the server might be misconfigured, Try turning on the "Disable java SNI extension" option and restart burp".

1. Project Options (far right) > SSL > use custom ciphers > select all the protocols
2. From same menu (just below), tick all three options in SSL Workarounds
3. User Options (far right) > Disable java SNI extension
