# List Shares


• List shares for every user you find (including standard accounts/database accounts etc.)
• If Microsft SQL is running on the system, try accessing shares as sa or mssql

## smbclient


smbclient -L 192.168.1.102 - List all available shares on the server
smbclient -L 192.168.1.102 -U <user> - List all shares available to a user

## smbmap


A samba share enumerator written by Shawn Evans.
VERY GOOD FOR LISTING/BROWSING SHARES THAT REQUIRE CREDENTIALS

smbmap -H 10.10.10.100 - Query all shares
smbmap -R [Folder] -H 10.11.0.4 - Recursive list
smbmap -r [Folder] -H 10.11.0.4 - Non recursive list

smbmap -u Finance -p Acc0unting -H 10.10.10.59 - Query with user/password
smbclient //10.10.10.59/ACCT -U Finance -s ‘ACCT’ -R - Recursively list files in the share. -s to specify the share

IF ATTACKING WINDOWS, try authenticating as Guest
smbmap -H 10.11.1.128 -u Guest
