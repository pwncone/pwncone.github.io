# Linux - Packages


• Linux systems are comprised of many packages
◇ internet browsers
◇ text editors
◇ media players
◇ etc.
• Packages are simply lots of files that have been compiled into 1
• Packages are managed via package managers, which install and maintain the software on your system

## Package Repositories


• Package Repositories are just a central storage location for packages
• There are tons of package repositories all over the internet (which contain different packages)
• A list of package repositories is stored in your system's /etc/apt/sources.list
◇ This file tells your machine which repositories to look in for software

## dpkg


• Used in to install .deb files
• Will install the package, but not its dependencies

dpkg -i some_deb_package.deb - install a package
dpkg -r some_deb_package.deb - remove a package
dpkg -l - list install packages

## apt


apt install package_name - install a package
apt remove package_name - remove a package
apt update; apt upgrade - find updates for packages and install them
apt show package_name - show info about installed package
