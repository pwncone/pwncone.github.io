# Web Concepts



## Identifying Client Browser & Platform


• Useful environment variables can be found in phpinfo.php page
◇ use getenv() to access them
◇ HTTP_USER_AGENT - identifies user's browser & OS

$_SERVER
https://www.php.net/manual/en/reserved.variables.server.php
• array containing info
◇ headers, paths, script locations
• just takes the headers etc. from the request (i think)
• phpinfo() will show you all of this stuff

e.g:
printf("%s <br>", $_SERVER['HTTP_USER_AGENT']); - print client user agent
printf("%s <br>", $_SERVER['REMOTE_ADDR']); - print client ip address

## HTML Forms


Any form element in an HTML page will be automatically available to your PHP scripts
• $_PHP_SELF refers to current PHP script (its name)
◇ i.e. the page will call the PHP code contained in itself

<html>
<body>
    <form action="<?php $_PHP_SELF ?>" method="POST">
        name: <input type="text" name="name"/>
        age: <input type="text" name="age"/>
        <input type="submit"/>
    </form>
</body>
</html>
 
<?php
    if ($_POST["name"] || $_POST["age"])
    {
        if (preg_match("/[^A-Za-z'-]/", $_POST["name"]))
            die("invalid name: alpha characters only");
        
        if (is_numeric($_POST["age"]) == FALSE)
            die("invalid age: numbers only");
        
        printf("hey %s <br>", $_POST["name"]);
        printf("you're %s years old :) <br>", $_POST["age"]);
    }
?>


## Browser Redirection


header() supplies raw HTTP headers to the browser
Example below is of modify the Location: header.
If you write a new URL into the Location: header, you will be redirected to that URL

<html>
<body>
    <p>visit a site:</p>
    <form action="<?php $_SERVER['PHP_SELF']?>" method="POST">
        <select name="location">
            <option value="http://youtube.com">youtube</option>
            <option value="http://google.com">google</option>
        </select>
        <input type="submit">
    </form>
</body>
</html>

<?php
    if ($_POST["location"])
    {
        $loc = $_POST["location"];
        header("Location: $loc");
    }
?>
