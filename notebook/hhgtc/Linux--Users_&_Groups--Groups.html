# Linux - User Management - Groups


• Groups are sets of users.
• Each gropu has a group name and an assosciated GUID - a group user id.
• Groups allow users to share files with other users in the same group.
• Groups are usually created based on teams e.g. marketing/web dev/business etc.

## GUIDs - Group IDs


• When you create a new user, they're also given their own group
◇ e.g. Bob's UID = 1001, Bob's GUID = 1001
• A user can be part of multiple groups
◇ e.g. Bob works in both the Marketing and Finance departments
▪ so Bob is added to both the Marketing and Finance groups
▪ this means that he can read any file created by anyone in those groups
• All groups have a GUID- Group ID
e.g. Marketing = 2004, Finance = 2005
