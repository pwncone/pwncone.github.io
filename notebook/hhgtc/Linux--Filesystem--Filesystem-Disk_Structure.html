# Linux - Filesystem Structure



## Boot block


• The boot block is located in the first few sectors of the filesystem
• It stores the information used to boot the operating system.
• Only 1 boot block is needed by the operating system.
• If you have multiple partitions, those partitions will also have boot blocks, but they won't be used

## Super block


• The super block sits just after the boot block
• It contains information about the filesystem
◇ size of the inode table
◇ size of the logical blocks
◇ size of the filesystem

## Inode table


• Inodes store information about files
• The inode table has an entry for each inode on your system

## Data blocks


• The data block is the last block on your system, and is where you actual data is :)
