# Found a strange script?



## Investigate


• Can you inject commands into it?
• Can any of the commands/code in the script be injected into?
• Can you write/modify/delete any of the files it's creating/manipulating/using?
• Are the commands in the script being run as root?
◇ If a tar command is being run as root, any files in that tar archive owned by root will still be root when they're extracted
▪ check hackthebox TartarSauce for an example
