# Ajenti


http://ajenti.org/
Ajenti is a control panel for remotely managing Linux systems.

## Get a shell


Examples
• hackthebox Luke

There's potentially 2 easy options to reach root from an Ajenti panel:
• change a user's password, change root's password, ssh in as the user and then switch user to root
• use Ajenti's Terminal tool to get an interactive web shell as root
