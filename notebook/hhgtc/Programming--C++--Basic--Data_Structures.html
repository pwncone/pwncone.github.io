#Data Structures


Data Structures allow you to combine data types of different kinds (as opposed to arrays which store multiple values of only 1 data type)

Structures are often used to keep records.
Say you want to a keep a record of all the books in your library, you would want to store information such as:
Title
Author
Book ID

Defining a structure


struct Books
{
    string title;
    string author;
    int book_id;
};


Declare a structure variable


struct Books book1;


You can also declare structure variables at the end of the structure definition
struct Books
{
    string title;
    string author;
    int book_id;
} book1, book2, book3;


Accessing structure members


To access any member of the structure you use the member access operator - .
#include <iostream>
#include <string>
using namespace std;

// define structure
struct Books
{
    string title;
    string author;
    int book_id;
};

int main()
{
    // declare structure variables
    struct Books book1;
    struct Books book2;

    // populate structure variable - book1
    book1.title = "The Pit and the Pendulum";
    book1.author = "Edgar Allan Poe";
    book1.book_id = 001;

    // populate structure variable - book2
    book2.title = "A Handful of Dust";
    book2.author = "Evelyn Waugh";
    book2.book_id = 002;

    // print out contents
    cout << "Book ID: " << book1.book_id << endl;
    cout << book1.title << endl; 
    cout << book1.author << endl;

    cout << endl;
    cout << "Book ID: " << book2.book_id << endl;
    cout << book2.title << endl; 
    cout << book2.author << endl;
    
    return 0;
}


Structures as Function Arguments


You can pass a structure as a function argument.

Create the function like so:
void printBookInfo(struct Books book)
And call the function:
printBookInfo(book1);

#include <iostream>
#include <string>
using namespace std;

// define structure for Books
struct Books
{
    string title;
    string author;
    int book_id;
};

// function to print out book info
void printBookInfo(struct Books book)
{
    cout << "\nBook ID: " << book.book_id << endl;
    cout << book.title << endl;
    cout << book.author << endl;
}

int main()
{
    // declare structure variables
    struct Books book1;
    struct Books book2;

    // populate structure variable - book1
    book1.title = "The Pit and the Pendulum";
    book1.author = "Edgar Allan Poe";
    book1.book_id = 001;

    // populate structure variable - book2
    book2.title = "A Handful of Dust";
    book2.author = "Evelyn Waugh";
    book2.book_id = 002;

    // print out contents of structure using function
    printBookInfo(book1);
    printBookInfo(book2);
    
    return 0;
}


Pointers to Structures


You can also define pointers to structures:
struct Books *structPtrBooks;

Using this, you can store the memory address of one of your declared structure variables
structPtrBooks = &book1;

To access the members of a structure using a pointer, use the -> operator
structPtrBooks ->title;

#include <iostream>
#include <string>
using namespace std;

// define structure for Books
struct Books
{
    string title;
    string author;
    int book_id;
};

// function to print out book info
void printBookInfo(struct Books book)
{
    cout << "\nBook ID: " << book.book_id << endl;
    cout << book.title << endl;
    cout << book.author << endl;
}

int main()
{
    // declare structure variables
    struct Books book1;
    struct Books book2;

    // populate structure variable - book1
    book1.title = "The Pit and the Pendulum";
    book1.author = "Edgar Allan Poe";
    book1.book_id = 001;

    // populate structure variable - book2
    book2.title = "A Handful of Dust";
    book2.author = "Evelyn Waugh";
    book2.book_id = 002;

    // POINTERS TO STRUCTURES
    // define pointer to Books
    struct Books *structPtrBooks;
    cout << structPtrBooks << endl;
    // retrieve memory address of book1 structure variable
    structPtrBooks = &book1;
    cout << structPtrBooks << endl;
    //access the members of Books structure
    cout << structPtrBooks ->title << endl;
    
    return 0;
}
