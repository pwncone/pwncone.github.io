# HTTP Methods


You can check what HTTP methods a web page allows by sending an OPTIONS request.
Sometimes you get interesting results, like being able to PUT a file.

## Testing & Finding Methods


### curl


curl -X OPTIONS http://192.168.1.177/test -i

-i - include http header
-X - request to make (GET, PUT, etc.)

### Nmap


nmap -p 80 --script http-methods 192.168.1.177

nmap -p 80 --script http-methods --script-args http-methods.url-path='/test' 192.168.1.177

## HTTP Methods


### PUT and DELETE


PUT can upload files.
DELETE can delete files.

#### PUT with Nmap
https://nmap.org/nsedoc/scripts/http-put.html
nmap -p 80 <ip> --script http-put --script-args http-put.url='/uploads/rootme.php',http-put.file='/tmp/rootme.php'

#### PUT with Metasploit
Use the http_put module.
