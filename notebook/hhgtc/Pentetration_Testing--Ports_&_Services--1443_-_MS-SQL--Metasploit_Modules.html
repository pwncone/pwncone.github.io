## Metasploit Modules


• All modules - https://www.offensive-security.com/metasploit-unleashed/hunting-mssql/
• Attack MSSQL with Metasploit - https://www.darkoperator.com/blog/2009/11/27/attacking-mssql-with-metasploit.html

### Modules


auxiliary/scanner/mssql/mssql_ping - ping database and get version
auxiliary/scanner/mssql/mssql_login - bruteforce login
auxiliary/admin/mssql/mssql_enum - enumerate the database
auxiliary/admin/mssql/mssql_exec - uses xp_cmdshell to execute system commands
auxiliary/admin/mssql/mssql_escalate_dbowner - escalate current user to database owner (usually sa)
exploit/windows/mssql/mssql_payload - uploads a payload to the system & gets shell (providing you have privileges)
