# Drupal


Drupal is an open-source content management framework written in PHP.
It provides a back-end framework for blogs, corporate, government sites etc.

## Find Drupal Version


You can find the Drupal version by visiting http://drupal-site.com/CHANGELOG.txt

## Credentials


### Credential locations


/var/www/html/sites/default/settings.php

### Default Credentials


admin / admin@123 - Source: https://www.drupal.org/project/yg_flew/issues/2955962

## Tools


There's quite a few Drupal vulnerability scanners out there.
Give them a try.

### droopescan


https://github.com/droope/droopescan

droopsescan will scan a Drupal site for vulns.
It takes a while to run.

droopescan scan drupal -u http://hawk.htb -t 32

### Nmap http-drupal-enum


This is an Nmap script that will enumerate Drupal.
https://nmap.org/nsedoc/scripts/http-drupal-enum.html
