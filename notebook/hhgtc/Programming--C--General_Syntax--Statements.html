# Statements


Statements are decision making.

## if Statements


if something happens, do this.
else if something else happens, do this instead.
else otherwise do this.

int age = 20;
int ticket_price;

if (age <= 4)
{
    ticket_price = 0;
}
else if (age > 4 && age <= 18)
{
    ticket_price = 5;
}
else
{
    ticket_price = 10;
}

## switch Statements


switch statements test a variable against a list of values.
• You can only test variables and values of the same type

###

default

case


You can have an optional default case which runs when none of the cases are true
• It must appear at the end of the switch
• Doesn't need a break

#include <stdio.h>
 
int main () 
{
   char grade = 'B';

   switch(grade)
   {
      case 'A':
         printf("Excellent! \n");
         break;
      case 'B':
         printf("Great! \n");
         break;
      case 'C':
         printf("You did alright. \n");
         break;
      case 'D':
         printf("Not great. \n");
         break;
      case 'F':
         printf("Better luck next time :/ \n");
         break;
      default:
         printf("Invalid grade. \n");
   }
   printf("Your scored a grade %c. \n", grade);
 
   return 0;
}
