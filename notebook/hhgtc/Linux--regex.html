# Linux- regex (Regular Expressions)


Regular expressions are a tool to help you select sections of text based on patterns.
You use characters like ^ and [ ] etc. to pick what you want.

This is our example text:
sally sells seashells
by the seashore

## Regular Expressions


^

- select beginning of line


^by would match the line by the seashore

$

- select end of line


seashore$
would match the line by the seashore

.

- Matching any single character


b.
would match by
s.
would match sally, sells, seashells, seashore

[ ]

- allow for multiple options


Brackets allow us to specify multiple characters to search for
d[iou]g
would match: dig, dog, dug

The anchor tag - ^ - when used in a bracket means anything except the characters within the bracket:
d[^i]g
would match: dog and dug but not dig

Brackets can also use ranges to increase the amount of characters you want to use.
d[a-c]g
will match patterns like dag, dbg, and dcg

Be careful though because brackets are case sensitive:
d[A-C]g
will match dAg, dBg and dCg but not dag, dbg and dcg
