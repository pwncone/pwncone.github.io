# Active Directory - Group Policy


Group Policy provides the ability to manage config changes centrally via Active Directory.

You can configure:
• security settings
• registry-based policy settings
• group policy preferences
◇ startup / shutdown / log-on + log-off scripts

It can be abused for:
• priv-esc
• backdoors / persistence

i.e. if you control group policy, you could push a backdoor (startup / log-on script) to all machines in a domain and pwn them all
