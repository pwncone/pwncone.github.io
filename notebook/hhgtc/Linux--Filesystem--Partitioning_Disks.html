# Linux - Partitioning


• Hard disks can be subdivided into partitions
◇ /dev/sda is the whole disk
◇ /dev/sda1 is the first partition on that disk
• Partitions are extremely useful for
◇ separating data
◇ and if you need different filesytem types on 1 disk
• The type of partitions depend on your partition table

## MBR and GPT - Partition Tables


### MBR - Master Boot Record


• MBR is the traditional partition table / old standard
• Can have primary, extended, and logical partitions
• Supports 4 primary partitions per disk
• Can only boot from drives less than 2 tb

Example - a partition using the MBR partitioning table (msdos)
Can see the primary, extended and logical paritions

### GPT - Guided Partion Table


• GPT is the new standard for disk partitioning
• Supports 128 partitions per disk
• Each partition has a globally unique ID (GUID)
◇ the GUID is used to define partitions and partition types
• Used (mostly) with UEFI based booting
