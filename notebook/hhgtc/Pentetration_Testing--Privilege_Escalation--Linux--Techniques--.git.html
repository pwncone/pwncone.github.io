# .git


You might find interesting bits of development inside a .git repository

Examples
• hackthebox DevOops

• git status - get status
• git log - check history of git repo (changes etc.)
◇ git log --name-only --oneline
 ▪ --name-only to list the files that changed in each commit
▪ --oneline to reduce space

• git diff 1422e5a d387abf - see the difference between 2 commits
◇ + denotes what's been added
◇ - denotes what's been removed
• git checkout d387abf -- resources/integration/authcredentials.key - checkout a commit
• git reset --hard 7ff507d - reset back to an old commit
