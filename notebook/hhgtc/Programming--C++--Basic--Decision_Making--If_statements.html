# if statements


if something happens, do this.
else if something else happens, do this instead.
else, just do this.

The

if

statement


The if statement allows us to check if a condition is true or false, and execute different code according to the result.
To check if 2 values are equal, use the == operator

int foo = 1;
int bar = 2;

if (foo < bar) {
    cout << "foo is smaller than bar." << endl;
}
if (foo == bar) {
    cout << "foo is equal to bar." << endl;
}


else if


else if is used after an if statement to check another condition within the same block of checks

int foo = 1;
int bar = 2;

if (foo < bar) {
    cout << "foo is smaller than bar." << endl;
} else if (foo > bar) {
    cout << "foo is greater than bar." << endl;
}


else


else is used when none of the if statements match. Think of it like ‘otherwise’ -
if foo > bar, do this, else if foo < bar do this, otherwise do this

int foo = 1;
int bar = 2;

} else if (foo > bar) {

Logical Operators



Check multiple expressions


You can use logical operators to check multiple expressions
&& = and
|| = or

int foo = 1;
int bar = 2;
int moo = 3;

// if foo is less than bar AND moo is more than bar
if (foo < bar && moo > bar) {
    cout << "foo is smaller than bar AND moo is larger than bar." << endl;
}

// if foo is less than bar OR moo is more than bar
if (foo < bar || moo > bar) {
    cout << "foo is smaller than bar OR moo is larger than bar." << endl;
}
