#About SQL


• SQL stands for Sructured Query Language
• It's used for managing databases.
◇ Databses contain information- usernames/password/emails/domains/documents etc.
• SQL datbases are structured like this:
◇ Database > Tables > Columns > Data
◇ Databases contain Tables > Tables contain Columns > Columns contain Data
◇ e.g
Database: Website Info
Table 1: Website logins
Column 1: username
Column 2: password
Column 3: email
Table 2: User details
Column 1: Name
Column 2: SSN
• Queries are how you retrieve info from the database. They look like this:
◇ SELECT <column> FROM database_name.table WHERE column='value';
◇ e.g. SELECT username FROM website_info.website_logins WHERE username='bobby';
