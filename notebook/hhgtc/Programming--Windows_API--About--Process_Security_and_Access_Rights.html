# Windows API - Process Security and Access Rights


https://docs.microsoft.com/en-gb/windows/win32/procthread/process-security-and-access-rights

User logs in → systems collects data to uniquely identify user → stores this data in an access token

• Access token - describes the security context of all processes associated with the user
• Security context of a process - is the set of credentials given to the process or user account that created the process
