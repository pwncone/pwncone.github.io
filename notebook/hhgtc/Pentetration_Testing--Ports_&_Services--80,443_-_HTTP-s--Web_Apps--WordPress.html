# WordPress


WordPress is a content management system written in PHP and paired with a MySQL or MariaDB database.
It makes creating and managing a website very easy because it offers pre-made templates for users to choose from, a backend admin panel for managing web pages and data, and offers plugins for extendability.

It's pretty common.
You'll find a lot of WordPress sites on the internet.

## Pages


These pages should be present if the website is a WordPress site.
/wp-admin
/wp-content
/wp-plugins

## Tools


### wpscan


wpscan is a WordPress scanning tool.
https://blog.sucuri.net/2015/12/using-wpscan-finding-wordpress-vulnerabilities.html

#### Options
-o output to a file
--disable-tls-checks can be useful if ssl cert isn't recognised/verified

#### Examples
wpscan -url http://website.com -enumerate -p

Enumerate top 10 usernames
wpscan --url <url> -e u
Enumerate top 100 usernames
wpscan -url <url> -e u[1-100]
Enumerate vulnerable plugins, vulnerable themes, top 10 users and site media
wpscan -url https://url.com -e vp,vt,u,m
Enumerate all plugins, all themes, top 10 users and site media
wpscan -url https://url.com -e ap,at,u,m

Enumerate all plugins PASSIVELY
wpscan -url https://url.com -e ap
Enumerate all plugins AGGRESSIVELY
wpscan -url https://url.com -e ap --plugins-detection aggressive

#### Errors
403 Request Is Forbidden
If you get 403 response code it's probably because they're using a WebApp firewall and are blocking wpscan.
To get around this, you can try using a random user agent.
wpscan -u https://url.com -r
