# Windows IIS


IIS - Internet Information Services - is a web server created by Microsoft that runs on Windows.

## Notes


IIS executes ASP and/or ASPx code.
• ASP = vbscript based code
• ASPx/ASP.NET = .NET based code

Microsoft IIS sites have a lot of potential file extensions.
• A list of IIS file extensions are available from the Microsoft docs
• https://docs.microsoft.com/en-us/previous-versions/2wawkw1c(v=vs.140)

## Tips


### gobuster


If you're gobuster-ing an IIS server, try searching for .asp,.aspx etc. extensions with your wordlist too

## IIS Versions


The version of IIS running is typically tied to the version of Windows that's running.
You can identify the IIS version from the Windows version, or vice-versa the Windows version from the IIS version.

Nmap will usually identify the IIS version for you.
