# Linux - Filesystem - Swap Space


• Swap space is used when phsyical RAM is full
• Inactive/idle memory pages from idle programs get moved into the swap space partitioned on your hard drive
◇ This frees up room in RAM
• On modern systems, configure 20% of amount of RAM as swap space
◇ e.g. 8gb RAM = 1.6GB swap space (i.e. just use 2GB)
◇ This is because we have more RAM nowadays, so don't need much swap space anymore
• In the old days, you would configure 2x memory size as swap space
