# Nuklear


A C alternative to ImGui.
Requires a graphics device to draw with - directx9, directx11, etc.
Here's all the backends: https://github.com/vurtun/nuklear/tree/master/demo
(dx9 and dx11 basically)

If you're external you'll need to create a transparent overlay window and a graphics device.
If you're internal, you'll need to connect Nuklear to the internal graphics device (dx9, dx11)
