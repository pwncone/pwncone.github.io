## Aimbot


### Yaw and Pitch


Pitch range is -89 - 89 (you can't look straight up or straight down)
Yaw range is -180 - 180

Don't exceed these ranges.
If you do, it's an instant ban!

## ESP


Player information (coordinates, etc.) is only updated when the server decides that the entity is close enough.
This is why ESPs bug out and you can't see across the entire map, but only when an enemy is close enough.
It does this using the bDormant flag:
https://guidedhacking.com/threads/csgo-bdormant-how-to-find-m_bdormant-offset-tutorial.11152/
