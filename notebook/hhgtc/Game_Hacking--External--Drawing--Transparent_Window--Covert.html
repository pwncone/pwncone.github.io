Don't create a child window of the game.
WS_EX_TOPMOST is detected by some anti cheat.
WS_VISIBLE is detected by some anti cheat.

## Make your window slightly larger than the game and offset


Anti-cheat will enumerate all windows.
If it finds a window on top of the game and/or is the exact same size as the game,
it will make a guess that this is a cheat window.

To counter this, you can make your window slightly bigger than the game
and then offset all of your drawing to sit on top of the game.
