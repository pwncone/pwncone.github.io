# User input


Any time that you take input from a user, you should verify that it's what you actually want.

## Get string from user


gets() isn't safe.
scanf() isn't buffer safe.

Use fgets().
• fgets() will append a newline to the inputted string

char name[20];
printf("Hey :) What's your name? ");
fgets(name, 20, stdin);

printf("Hey %s :) \n", name);


## Get int/float etc. from user


Use scanf().
• scanf() stops taking input after a space

int age;
printf("Hey :) How old are you? ");
scanf("%d", &age);

printf("You're %d years old \n", age);
