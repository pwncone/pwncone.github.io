# Authentication


You can try testing the Authentication mechanism on the website.

## Cookies


Log in / register and inspect at the cookie you receive when logged in (via Burp or browser tools or whatever)

• Try changing the cookie's value
◇ Example - from Cookie: auth=billy to Cookie: auth=admin
• Google the cookie
• Try to crack the cookie
◇ Example #1
▪ Log in as user1
▪ Receive Cookie: auth=24c9e15e52afc47c225b757e7bee1f9d
▪ 24c9e15e52afc47c225b757e7bee1f9d is md5 hash of user1
▪ md5 hash "admin" and submit as cookie -> might get logged in as admin
