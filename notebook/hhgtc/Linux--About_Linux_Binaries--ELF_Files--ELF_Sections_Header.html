# ELF Sections Header


The ELF Sections Header defines the location of all the sections in the file.
Sections contain program data & information.

For executables, there are 4 main sections
.text - the program's instructions (the actual code/what gets executed)
.data - initialised data that's writable (int d = 4)
.rodata - read-only initialiased data (constants? - const int f = 6)
.bss - unitialised data that's writeable (int i)

The section header can be read with readelf -S <binary>

To find the addresses and offsets of the sections when they're loaded into memory, you have to read the program header.
