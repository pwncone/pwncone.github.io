# Node.js


Node.js can also spawn web servers.

The webserver lives on port 80.
The debugger lives on port 9229.

## Tips


• Proxy your traffic through Burp and watch the requests when browsing to a page
◇ You might see /api requests or something similar - this is where the data is being retrieved from
◇ Used in hackthebox Node

## Recon


# Find main app file used by the Node.js server
curl http://$ip:9229/json/list

# Connect to debug console
node-inspect $ip:9229
    # List the number of scripts loaded (including node-internals)
    debug> scripts(true)
