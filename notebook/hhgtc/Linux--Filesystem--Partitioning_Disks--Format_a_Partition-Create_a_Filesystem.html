# Linux - Partitions - Format a Partition/Create a Filesystem


After partitioning, you format the partition with a filesystem (e.g. ext4, NTFS etc.)

sudo mkfs -t ext4 /dev/sdb2

lsblk -f - list existing file systems
◇ sda/sdb = physical disks
◇ sda1/sdb2 = partitions on those disks

1. Mounted file systems must be unmounted before creating a new file system on that partition
unmount /dev/sda1
2. Make a new file system:
sudo mkfs.ext4 /dev/sda1
3. Mount the file system:
mount /dev/sda1 /
