# Constants


• constants can't change
• always uppercase
• have to define using define() function
◇ define("MINSIZE", 50);
• retrieve value using name or with constant()
◇ echo MINSIZE;
◇ echo constant("MINSIZE");
• only
◇ bool
◇ int
◇ float
◇ string
▪ can be constants

## Predefined Constants


PHP has a large number of predefined constants you can use

__LINE__ - the current line number in file
__FILE__ - full path & filname of current file
__FUNCTION__ - returns function name as it was declared
__CLASS__ - class name
__METHOD__ - class method name
