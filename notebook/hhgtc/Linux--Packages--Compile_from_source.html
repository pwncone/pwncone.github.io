# Linux - Packages - Compile from Source


Sometimes you'll need to compile code from source :/

1. sudo apt install build-essential - install the tools for compiling from source
2. tar -xzvf package.tar.gz - extract the contents of the package file (most likely a .tar.gz file)
3. Look at the README or INSTALL file inside the package - there might be specific installation instructions
4. Inside the package contents will be a configure script - this script checks for dependencies on your system - ./configure
5. Instead of using make to compile the code (leaves a bunch of junk on your comp), use checkinstall - sudo checkinstall
1) This builds a .deb package and installs it - this makes it easier to remove the package later
