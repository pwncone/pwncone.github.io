Data Directories


All of the possible data directories are listed here (towards the bottom):
https://docs.microsoft.com/en-us/windows/win32/api/winnt/ns-winnt-image_optional_header32

Some interesting ones:
IMAGE_DIRECTORY_ENTRY_BASERELOC - Base Relocations
IMAGE_DIRECTORY_ENTRY_EXPORT - Export Directory
IMAGE_DIRECTORY_ENTRY_IMPORT - Import Directory
