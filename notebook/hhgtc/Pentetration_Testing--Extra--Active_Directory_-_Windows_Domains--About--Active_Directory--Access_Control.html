# Active Directory - Access Control


Access Control (clue is in the name) defines who has access to stuff.

Active Directory is made up of objects.
Each Active Directory Object has a Security Descriptor - a SID.

Each SID contains:
• Owner - who owns this Active Directory Object
• A SACL - System ACL - which logs success/failure messages when this object is accessed
• A DACL - Discretionary ACL - which is a list of Access Control Entries - ACEs
◇ ACE #1
◇ ACE #2
◇ ACE #3
◇ etc.

Each ACE - Access Control Entry - contains:
• Permission Type - usually Allow or Disallow
• Principal - who has these permissions (Administrators, SYSTEM, Everyone, etc.)
• Access - what access they have (Special, Full Control, etc.)
• Object - what object these users have the set permissions on (usually this/the current object)
